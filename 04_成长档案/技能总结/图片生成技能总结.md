## 一、核心技能

### 1️⃣ Infographic Creator（信息图生成）⭐推荐

**技能位置**：`/root/.openclaw/skills/infographic-creator`

**适用场景**：
- 工程融合图（树状结构）
- 方法论展示（层级关系）
- 成果总结（数据可视化）

**使用方法**：
```html
<!DOCTYPE html>
<html lang="zh-CN">
<head>
  <meta charset="utf-8">
  <title>标题 - Infographic</title>
  <style>
    body { margin: 0; padding: 20px; background: #1a1a2e; }
    #container { width: 100%; height: 100vh; }
  </style>
</head>
<body>
<div id="container"></div>
<script src="https://unpkg.com/@antv/infographic@latest/dist/infographic.min.js"></script>
<script>
const infographic = new AntVInfographic.Infographic({
  container: '#container',
  width: '100%',
  height: '100%',
});
document.fonts?.ready.then(() => {
  infographic.render(`
infographic hierarchy-tree-tech-style-badge-card
data
  title 标题
  desc 描述
  root
    label 根节点
    children
      - label 子节点1
        children
          - label 叶节点
      - label 子节点2
theme
  palette #0066CC #00AA88 #FFD700
`);
});
</script>
</body>
</html>
```

**模板选择**：
| 模板 | 用途 |
|------|------|
| `hierarchy-tree-tech-style-badge-card` | 树状结构，科技风格 |
| `hierarchy-mindmap-level-gradient-compact-card` | 思维导图 |
| `list-grid-badge-card` | 列表网格 |
| `sequence-steps-simple` | 流程步骤 |
| `compare-swot` | SWOT分析 |

**色彩搭配**：
```
蓝 #0066CC - 专业稳重
绿 #00AA88 - 成长希望
金 #FFD700 - 价值精华
蓝浅 #60A5FA - 辅助色
橙 #F59E0B - 点缀色
```

### 2️⃣ SVG + ImageMagick（图标生成）⭐推荐

**适用场景**：
- 工程图标
- LOGO设计
- 简单图形

**使用方法**：
```bash
# 1. 创建SVG文件
cat > 图标.svg << 'EOF'
<svg xmlns="http://www.w3.org/2000/svg" viewBox="0 0 400 400" width="400" height="400">
  <defs>
    <linearGradient id="bgGrad" x1="0%" y1="0%" x2="100%" y2="100%">
      <stop offset="0%" style="stop-color:#0A1929;stop-opacity:1" />
      <stop offset="100%" style="stop-color:#1E3A8A;stop-opacity:1" />
    </linearGradient>
  </defs>
  <rect width="400" height="400" fill="url(#bgGrad)" rx="20"/>
  <!-- 更多SVG元素 -->
</svg>
EOF

# 2. 转换为PNG
convert 图标.svg 图标.png
```

**优势**：
- 矢量图，无损缩放
- 代码可控，精确设计
- 转换快速，一行命令

## 二、出图流程标准

### 标准流程

```
Step 1: 确定需求
        ↓
        图标？融合图？纪念图？

Step 2: 选择方法
        ↓
        图标 → SVG + ImageMagick
        融合图 → Infographic Creator
        复杂图 → HTML + Canvas

Step 3: 设计内容
        ↓
        确定结构、配色、文字

Step 4: 生成文件
        ↓
        write HTML/SVG文件

Step 5: 转换输出
        ↓
        SVG → PNG（convert命令）
        HTML → 在浏览器中查看

Step 6: 发送给用户
        ↓
        <qqimg>图片路径</qqimg>
```

## 六、百度搜索技能（已安装）⭐

**技能位置**：`/root/.openclaw/workspace/06_技能库/baidu-search`

**使用方法**：
```bash
python3 skills/baidu-search/scripts/search.py '{"query":"搜索关键词"}'
```

**参数说明**：
| 参数 | 说明 |
|------|------|
| `query` | 搜索关键词（必填） |
| `search_recency_filter` | 时间过滤：`week`, `month`, `semiyear`, `year` |
| `resource_type_filter` | 资源类型：web, video, image |

**示例**：
```bash
# 基础搜索
python3 /root/.openclaw/workspace/06_技能库/baidu-search/scripts/search.py '{"query":"AI营销案例"}'

# 时间过滤
python3 /root/.openclaw/workspace/06_技能库/baidu-search/scripts/search.py '{"query":"最新新闻","search_recency_filter":"week"}'
```

**注意**：需要配置 `BAIDU_API_KEY` 环境变量
