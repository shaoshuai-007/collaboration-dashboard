## 一、配色规范

### 1.1 主色调

| 颜色名称 | RGB值 | HEX值 | 用途 |
|---------|-------|-------|------|
| 钢蓝色（主色） | RGB(70, 130, 180) | #4682B4 | 主要图形、标题装饰 |
| 深蓝色（主色深） | RGB(30, 60, 114) | #1E3C72 | 封面渐变、深色背景 |
| 珊瑚红（点缀色） | RGB(255, 107, 107) | #FF6B6B | 重点强调、第一顺位 |
| 矢车菊蓝（辅色） | RGB(100, 149, 237) | #6495ED | 辅助图形、第三顺位 |

### 1.2 中性色

| 颜色名称 | RGB值 | HEX值 | 用途 |
|---------|-------|-------|------|
| 深灰 | RGB(64, 64, 64) | #404040 | 正文文字 |
| 中灰 | RGB(128, 128, 128) | #808080 | 辅助文字 |
| 浅灰 | RGB(245, 247, 250) | #F5F7FA | 页面背景 |
| 白色 | RGB(255, 255, 255) | #FFFFFF | 卡片背景、文字反白 |
| 黑色 | RGB(33, 33, 33) | #212121 | 标题文字 |

### 1.3 配色应用规则

#### 规则1：顺位配色
- **第一顺位**：珊瑚红 - 用于最重要的内容
- **第二顺位**：钢蓝色 - 用于次重要的内容
- **第三顺位**：矢车菊蓝 - 用于辅助内容

#### 规则2：对比配色
- **深色背景**：白色文字
- **浅色背景**：深灰文字
- **强调内容**：珊瑚红背景 + 白色文字

#### 规则3：渐变配色
- **封面渐变**：深蓝色 → 钢蓝色（角度90度）
- **装饰元素**：珊瑚红点缀

## 二、图标尺寸规范

### 2.1 标准尺寸

| 级别 | 宽度（英寸） | 高度（英寸） | 用途 |
|------|------------|------------|------|
| 特大 | 3.1 | 0.8 | 重点内容、主要模块 |
| 大 | 3.1 | 0.6-0.7 | 重要内容、业务场景 |
| 中 | 2.0 | 0.75 | 常规内容、标准图标 |
| 小 | 1.5 | 0.6-0.65 | 辅助内容、次要信息 |
| 超小 | 1.0 | 0.55 | 密集内容、补充信息 |

### 2.2 尺寸应用场景

#### 封面页
- 主标题：无图标，纯文字
- 副标题：无图标，纯文字
- 装饰线：宽度3.333英寸，高度0.03英寸

#### 目录页
- 序号框：0.65 × 0.65英寸
- 标题：纯文字
- 描述：纯文字

#### 内容页
- **重点举措标题**：纯文字
- **指标卡片**：宽度根据页面宽度均分，高度0.7倍于宽度

#### 架构页
- **层名称**：1.15 × 0.75英寸
- **层元素**：1.6 × 0.41英寸

#### 能力页
- **业务场景**：3.1 × 0.6英寸
- **核心能力**：与业务场景一致
- **技术流程**：2.0 × 0.55英寸

#### 问题分析页
- **问题表象**：2.0 × 0.75英寸
- **根因分析**：2.0 × 0.75英寸
- **解决方案**：2.0 × 0.75英寸

#### 系统集成页
- **API网关**：2.5 × 0.8英寸
- **上游系统**：2.5 × 0.65英寸
- **下游系统**：2.0 × 0.65英寸

### 2.3 尺寸一致性原则

#### 原则1：同类型图标尺寸一致
- 同一页面内，相同类型的图标尺寸必须一致
- 例如：问题分析页的"效率低、体验差、知识获取难"都是2.0 × 0.75

#### 原则2：层级关系清晰
- 上层内容图标 > 下层内容图标
- 主要内容图标 > 辅助内容图标

#### 原则3：视觉平衡
- 每行图标总宽度应接近页面宽度
- 避免图标过大或过小导致视觉失衡

## 三、布局规范

### 3.1 页面边距

| 区域 | 边距（英寸） |
|------|------------|
| 左边距 | 0.5-0.8 |
| 右边距 | 0.5-0.8 |
| 上边距 | 0.4-0.6 |
| 下边距 | 0.4-0.6 |

### 3.2 内容间距

| 类型 | 间距（英寸） |
|------|------------|
| 标题与正文 | 0.4 |
| 段落之间 | 0.2 |
| 图标之间（横向） | 0.15-0.2 |
| 图标之间（纵向） | 0.1-0.15 |
| 图标与描述文字 | 0.05-0.1 |

### 3.3 对齐规范

#### 水平对齐
- **标题行**：左对齐，从0.85英寸开始
- **正文内容**：左对齐，从0.8英寸开始
- **图标组**：居中对齐或左对齐
- **单个居中元素**：水平居中于页面

#### 垂直对齐
- **图标内的文字**：垂直居中（MSO_ANCHOR.MIDDLE）
- **图标组**：顶部对齐
- **多行内容**：顶部对齐

### 3.4 布局模式

#### 模式A：左右分栏（常用）
```
| 左侧（40%）| 右侧（60%）|
| 重点举措   | 图示/数据  |
```

#### 模式B：上下分层
```
| 标题区      |
| 内容区      |
| 图表区      |
```

#### 模式C：网格式
```
| 图标1 | 图标2 | 图标3 |
| 图标4 | 图标5 | 图标6 |
```

#### 模式D：流式
```
| 图标1 |
   ↓
| 图标2 |
   ↓
| 图标3 |
```

## 五、视觉效果规范

### 5.1 阴影效果

#### 参数设置
- **模糊半径**：50800 EMU（约0.05英寸）
- **距离**：38100 EMU（约0.04英寸）
- **方向**：2700000（向下）
- **透明度**：40%

#### 应用场景
- 所有圆角矩形图标
- 所有卡片式元素
- 不应用于：装饰线、纯文字

### 5.2 渐变效果

#### 线性渐变
- **角度**：90度（从上到下）
- **停止点**：2个
- **颜色1**：主色深
- **颜色2**：主色

#### 应用场景
- 封面背景
- 结束页背景

### 5.3 圆角效果

#### 圆角矩形
- 使用`MSO_SHAPE.ROUNDED_RECTANGLE`
- 自动圆角，无需手动设置

## 六、元素规范

### 6.1 装饰元素

#### 顶部装饰线
- **位置**：页面顶部，宽度全宽
- **尺寸**：13.333英寸 × 0.15英寸
- **颜色**：主色或点缀色

#### 标题装饰线
- **位置**：标题左侧
- **尺寸**：0.15英寸 × 0.6英寸
- **颜色**：主色

#### 分隔线
- **位置**：内容区域之间
- **尺寸**：宽度根据内容，高度0.03英寸
- **颜色**：点缀色

### 6.2 箭头元素

#### 下箭头
- **尺寸**：宽度0.15-0.25英寸，高度0.18-0.4英寸
- **颜色**：中灰
- **用途**：表示流程方向

### 6.3 指标卡片

#### 尺寸计算
```
卡片宽度 = (页面可用宽度 - (卡片数量-1) × 间距) / 卡片数量
卡片高度 = 卡片宽度 × 0.7
```

#### 内容布局
```
| 数值（大号加粗）|
| 标签（小号常规）|
```

## 八、质量检查清单

### 8.1 配色检查

- [ ] 主色是否为钢蓝色 RGB(70, 130, 180)
- [ ] 点缀色是否为珊瑚红 RGB(255, 107, 107)
- [ ] 背景色是否为浅灰 RGB(245, 247, 250)
- [ ] 文字色是否为深灰 RGB(64, 64, 64)
- [ ] 渐变色是否为深蓝 → 钢蓝

### 8.2 图标尺寸检查

- [ ] 同类型图标尺寸是否一致
- [ ] 重点内容图标是否大于辅助内容
- [ ] 图标是否有阴影效果
- [ ] 图标是否为圆角矩形

### 8.3 布局检查

- [ ] 页面边距是否统一
- [ ] 内容间距是否合理
- [ ] 对齐方式是否一致
- [ ] 整体布局是否平衡

### 8.4 字体检查

- [ ] 字体是否为微软雅黑
- [ ] 标题字号是否为24pt
- [ ] 正文字号是否为12pt
- [ ] 字体颜色是否规范

### 8.5 视觉效果检查

- [ ] 阴影效果是否统一
- [ ] 渐变效果是否正确
- [ ] 装饰元素是否规范
- [ ] 整体视觉是否协调

## 九、使用示例

### 示例1：生成标准内容页

```python
# 标准内容页生成
def generate_content_page(slide, title, subtitle, target, measures, right_content):
    # 背景
    add_rect(slide, 0, 0, 13.333, 7.5, NEUTRAL_LIGHT)
    
    # 顶部装饰线
    add_rect(slide, 0, 0, 13.333, 0.15, PRIMARY)
    
    # 标题区
    add_header(slide, title, subtitle)
    
    # 目标行
    add_target_line(slide, target)
    
    # 重点举措标题
    add_measure_title(slide)
    
    # 左侧：举措内容
    add_measure_content(slide, measures)
    
    # 右侧：图/数据
    add_right_content(slide, right_content)
```

### 示例2：生成标准图标

```python
# 标准图标生成
def generate_standard_icon(slide, left, top, text, priority="normal"):
    # 根据优先级确定颜色
    if priority == "high":
        color = ACCENT  # 珊瑚红
    elif priority == "normal":
        color = PRIMARY  # 钢蓝色
    else:
        color = SECONDARY  # 矢车菊蓝
    
    # 标准尺寸：2.0 × 0.75
    add_shadow_box(slide, left, top, 2.0, 0.75, color, text, 10, WHITE)
```
