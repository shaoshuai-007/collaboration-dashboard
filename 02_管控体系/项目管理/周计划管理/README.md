
## 🎯 管理原则

### 1. 周计划管控
- **周一**：确认本周重点任务，分解到天
- **周三**：进度检查，风险识别
- **周五**：周总结，闭环检查

### 2. 日计划执行
- **每日晨会**：确认当日重点，责任人明确
- **每日晚报**：进展更新，问题上报
- **日清日结**：当天任务当天闭环

### 3. 闭环管理
- **计划**：目标明确、可衡量
- **执行**：责任人明确、时间节点清晰
- **检查**：进度可见、风险预警
- **处理**：问题闭环、经验沉淀

## 📊 任务状态定义

| 状态 | 含义 | 标识 |
|------|------|:----:|
| 未开始 | 尚未启动 | ⚪ |
| 进行中 | 正在执行 | 🔵 |
| 已完成 | 任务完成 | ✅ |
| 有风险 | 存在阻塞 | 🔴 |
| 已挂起 | 暂时搁置 | ⏸️ |

## ⚠️ 风险预警机制

| 级别 | 触发条件 | 处理方式 |
|:----:|---------|---------|
| 🔴 高 | 延期超3天/阻塞严重 | 立即升级汇报 |

| 🟢 低 | 按计划进行 | 周报跟踪 |
