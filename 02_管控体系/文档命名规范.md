## 命名规则

```
日期-类型-项目名称-版本.扩展名

示例：
2026-03-26-需求-湖北电信AI配案-V1.0.md
2026-03-26-架构-Token采集系统-V2.0.docx
2026-03-26-设计-数据库表结构-V1.0.xlsx
```

## 检查脚本

```bash
# 检查命名规范
find /root/.openclaw/workspace -name "*.md" -o -name "*.docx" -o -name "*.xlsx" | grep -v "^[0-9]\{4\}-[0-9]\{2\}-[0-9]\{2\}" 
```
