## 📝 入库标准

### 文件命名规范

| 类型 | 命名格式 | 示例 |
|------|---------|------|
| 知识文档 | `主题_版本.md` | `情报价值评估模型_v2.0.md` |
| 商机文档 | `项目名称_类型.md` | `AI智能配案_跟进记录.md` |
| 方案文档 | `项目名称_方案V版本.md` | `湖北AI配案_方案V3.0.md` |
| 会议纪要 | `日期_主题_纪要.md` | `2026-03-11_电渠项目_纪要.md` |

### 目录维护原则

1. **保持整洁**：文件放入对应目录，不乱放
2. **命名规范**：统一格式，便于检索
3. **定期清理**：过期文件移入历史归档
4. **持续优化**：根据使用反馈调整结构
