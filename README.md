# 智能体协作平台 (Collaboration Dashboard)

> 九星智囊团 - 多智能体协作决策平台

## 项目简介

智能体协作平台是一个多角色AI协作决策系统，通过多智能体辩论和质疑机制，帮助团队快速达成结构化决策方案。

### 核心目标

1. **替代/辅助人工前期决策**：工程启动前，自动完成需求拆解、技术选型、风险评估、成本测算等环节的多角色论证
2. **输出结构化决策方案**：将模糊需求转化为可落地、可评估、可追溯的工程方案
3. **降低决策风险**：通过多角色自主辩论、互相反驳，提前暴露方案漏洞

## 功能特性

### 🎭 多角色协作

| Agent | 角色 | 职责 |
|:-----:|------|------|
| 🌸 采薇 | 需求分析师 | 需求拆解、用户故事、验收标准 |
| 🧵 织锦 | 架构设计师 | 技术选型、架构设计、扩展性评估 |
| 🏗️ 筑台 | 售前工程师 | 成本估算、资源需求、方案包装 |
| 🎨 呈彩 | 方案设计师 | UI设计、用户体验、视觉呈现 |
| 📐 工尺 | 系统设计师 | 接口设计、模块划分、详细设计 |
| ⚖️ 玉衡 | 项目经理 | 进度计划、风险管控、里程碑管理 |
| 📚 折桂 | 资源管家 | 知识管理、资源调度、信息采集 |
| 🌀 扶摇 | 总指挥 | 统筹协调、决策拍板、方向把控 |
| 🌿 南乔 | 用户助手 | 用户交互、任务分发、结果推送 |

### 💬 质疑机制

- 质疑检测：关键词识别
- 质疑分类：技术/成本/进度/风险/需求
- 质疑链路：前端实时展示质疑关系图
- 质疑响应：被质疑Agent自动生成回应
- 质疑过滤：按Agent筛选质疑记录

### 📊 共识度计算

- 基础共识度：70%
- 每条质疑扣分：5%
- 最低共识度：30%

### 📝 会议纪要自动生成

- 项目概况
- 讨论内容
- 关键决策
- 风险提示
- 下一步行动

## 技术栈

- **后端**：Python 3.11 + Flask
- **前端**：HTML5 + CSS3 + JavaScript
- **AI模型**：千帆大模型 (qianfan-code-latest)
- **数据存储**：SQLite

## 使用场景

### 场景1：需求分析

输入：客户需求描述
输出：结构化需求文档（用户故事、验收标准、业务规则）

### 场景2：技术选型

输入：项目需求
输出：技术选型方案（架构设计、技术栈、扩展性评估）

### 场景3：成本估算

输入：项目范围
输出：成本估算报告（人力、技术、时间、风险成本）

### 场景4：风险评估

输入：项目计划
输出：风险分析报告（风险识别、影响评估、应对策略）
