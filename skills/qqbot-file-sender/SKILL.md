# QQ文件发送技能

## ⚠️ 当前状态：QQ文件发送受限

**由于QQ机器人API限制，文件直接发送功能暂时不可用。但南乔已为您准备了两套替代方案！**

---

## 方案1：邮箱发送（推荐 ✅）

**已实现**：文件自动发送到少帅邮箱（szideaf7@163.com）

**使用方法**：
- 只需告诉南乔"发给你"或"发到邮箱"
- 南乔自动调用send-email技能发送

**示例**：
```
少帅，报告已发送到您邮箱！
📧 接收地址：szideaf7@163.com
📎 附件：中国电信vs中国移动_2025年业绩对比报告.md
```

---

## 方案2：发送文件链接

如果需要通过QQ发送，可以生成文件下载链接：

1. 将文件上传到可访问的服务器
2. 发送链接给用户

## 快速使用指南

### 发送报告/文档

当少帅要求发送文件时：

```
少帅，已发送到您邮箱！
📧 szideaf7@163.com
📎 附件：xxx.md
```

### 发送图片

使用 `<qqimg>` 标签：

```
<qqimg>/root/.openclaw/workspace/xxx.png</qqimg>
```

---

## 技术说明

- **文件大小限制**：邮箱附件建议 < 20MB
- **支持格式**：.md, .doc, .pdf, .xlsx, .txt 等
- **自动转换**：Markdown会自动转为HTML格式发送
