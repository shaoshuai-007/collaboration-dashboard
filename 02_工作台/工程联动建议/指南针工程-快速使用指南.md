### 📚 折桂（资源管家）

| 指令 | 说明 | 输出 |
|------|------|------|
| `配色方案` | 查询三原色定理配色 | 配色说明 |
| `审核标准` | 查询质量审核标准 | 审核清单 |
| `方法论` | 查询工作方法论 | 方法说明 |

**示例**：
```
用户：配色方案是什么？
折桂：三原色定理（60-30-10）：
      
      主色（60%）：电信红 #C93832
      - 用于：标题、重点、核心元素、封面背景
      
      辅色（30%）：深蓝 #006EBD
      - 用于：次要标题、图表、系统元素
      
      中性色（10%）：灰色 #595959
      - 用于：正文文字、背景、分隔线
      
      禁止：使用超过3种主色，避免绿、橙、紫混用
```

## 三、工作流程

### 标准流程（推荐）

```
步骤1：需求文档    → 采薇生成需求文档
步骤2：思维导图    → 织锦生成分析图
步骤3：方案举措    → 筑台生成方案表
步骤4：PPT         → 呈彩生成方案PPT
───────── 人工评审 ─────────
步骤5：详细设计    → 工尺生成设计文档
步骤6：项目管控    → 玉衡生成项目计划
```

### 一键全流程

```
全流程生成【项目名称】
```

扶摇会自动执行步骤1-6，在步骤4后暂停等待评审。

## 四、常见问题

### Q1：如何修改配色方案？

修改 `/root/.openclaw/skills/compass-shared/references/配色方案.md`

### Q2：如何自定义PPT模板？

修改 `/root/.openclaw/skills/compass-ppt/scripts/generate_ppt.py`

### Q3：输出文件在哪里？

默认保存在 `/root/.openclaw/workspace/output/`

### Q4：如何查看项目进度？

发送 `项目管控` 查看甘特图和进度
