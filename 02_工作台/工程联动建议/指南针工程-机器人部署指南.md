## 二、部署方式

### 2.1 会话隔离模式

使用OpenClaw的会话隔离功能，为每个技能创建独立会话：

```yaml
# 在OpenClaw配置中添加
sessions:
  compass-needdoc:
    agent: main
    skills:
      - compass-needdoc
      - compass-shared
    triggers:
      - "需求文档"
      - "需求分析"
      
  compass-mindmap:
    agent: main
    skills:
      - compass-mindmap
      - compass-shared
    triggers:
      - "思维导图"
      - "分析图"
      
  # ... 其他技能
```

### 2.2 主控机器人模式

创建主控机器人（扶摇），协调所有技能：

```yaml
sessions:
  compass-coordinator:
    agent: main
    skills:
      - compass-coordinator
      - compass-needdoc
      - compass-mindmap
      - compass-solution
      - compass-ppt
      - compass-design
      - compass-project
      - compass-shared
    triggers:
      - "全流程"
      - "一键生成"
```

## 三、自动化配置

### 3.1 定时任务

使用OpenClaw的cron功能，设置定时任务：

```bash
# 每周一9:00提醒项目进度
openclaw cron add \
  --name "周报提醒" \
  --cron "0 9 * * 1" \
  --tz "Asia/Shanghai" \
  --session isolated \
  --message "📊 本周项目进度汇报时间到！" \
  --deliver \
  --channel qqbot \
  --to "8236C3DA8CF6F5DF6FEE66D42ADAAE97"
```

### 3.2 工作流触发

当某个步骤完成后，自动触发下一步：

```python
# 在步骤完成后发送通知
def on_step_complete(step_num, output_path):
    message = f"""
✅ 步骤{step_num}已完成

输出文件：{output_path}

回复"继续"执行下一步
"""
    send_message(channel="qqbot", to=openid, message=message)
```
