## 一、配色体系

### 1.1 主色系

| 颜色名称 | RGB值 | 十六进制 | 使用场景 |
|---------|-------|----------|----------|
| **电信红** | 201, 56, 50 | #C93832 | 主标题、重点强调、核心元素 |
| **深红** | 172, 0, 0 | #AC0000 | 标题装饰、边框 |

### 1.2 辅色系

| 颜色名称 | RGB值 | 十六进制 | 使用场景 |
|---------|-------|----------|----------|
| **橙色** | 237, 125, 49 | #ED7D31 | 副标题、流程第二步、次要强调 |
| **蓝色** | 0, 110, 189 | #006EBD | 技术元素、流程第三步、系统 |
| **浅蓝** | 79, 129, 189 | #4F81BD | 辅助元素、背景装饰 |
| **深蓝** | 31, 73, 125 | #1F497D | 深色背景、文字反白 |

### 1.3 中性色系

| 颜色名称 | RGB值 | 十六进制 | 使用场景 |
|---------|-------|----------|----------|
| **深灰** | 89, 89, 89 | #595959 | 正文文字、说明文字 |
| **灰色** | 166, 166, 166 | #A6A6A6 | 辅助文字、分隔线 |
| **浅灰** | 240, 240, 240 | #F0F0F0 | 背景色、卡片底色 |
| **白色** | 255, 255, 255 | #FFFFFF | 主背景、文字反白 |

### 1.4 功能色系

| 颜色名称 | RGB值 | 十六进制 | 使用场景 |
|---------|-------|----------|----------|
| **绿色** | 84, 130, 53 | #548235 | 成功、完成、正向指标 |
| **青色** | 0, 128, 128 | #008080 | 信息、提示 |

### 1.5 配色原则

```
规则1：每页不超过3种主色
规则2：电信红为主色，占60%以上
规则3：橙色、蓝色为辅色，各占15-20%
规则4：灰色系用于文字和背景
规则5：避免大面积使用深色背景
```

├─────────────────────────────────────────────────┤

## 四、图模板决策树

### 4.1 决策规则

```
内容类型判断：
├── 流程展示类
│   ├── 技术流程 → 横向流程图（6步，箭头连接）
│   ├── 业务流程 → 纵向流程图（4步，自上而下）
│   └── 交互流程 → 泳道流程图（多角色协作）
│
├── 架构展示类
│   ├── 系统架构 → 六层分层架构图
│   ├── 组织架构 → 树形结构图
│   └── 系统集成 → 星型拓扑图（中心辐射）
│
├── 指标展示类
│   ├── 单指标 → 仪表盘图（圆环+数字）
│   ├── 多指标对比 → 柱状图/雷达图
│   └── 趋势展示 → 折线图/时间轴
│
├── 价值展示类
│   ├── 价值链 → 价值链图（4块并列）
│   ├── 对比展示 → 左右对比表格
│   └── 创新亮点 → 创新矩阵图（5块并列）
│
└── 案例展示类
    ├── 成功案例 → 案例卡片（3个并列）
    └── 经验总结 → 要点列表
```

### 4.2 图模板尺寸规范

| 图模板类型 | 宽度 | 高度 | 位置 |
|-----------|------|------|------|
| 横向流程图 | 6.0英寸 | 1.0英寸 | 右侧中部 |
| 分层架构图 | 5.0英寸 | 4.5英寸 | 右侧居中 |
| 星型拓扑图 | 6.0英寸 | 4.0英寸 | 右侧居中 |
| 仪表盘图 | 5.0英寸 | 3.0英寸 | 右侧上部 |
| 价值链图 | 12.0英寸 | 2.0英寸 | 页面下方 |
| 创新矩阵图 | 12.0英寸 | 3.0英寸 | 页面下方 |

## 六、质量检查清单

### 6.1 视觉层面

- [ ] 配色是否统一（电信红为主色）
- [ ] 字体是否统一（微软雅黑）
- [ ] 字号是否规范（标题24、正文12）
- [ ] 间距是否一致

### 6.2 内容层面

- [ ] 每页是否有明确的建设目标
- [ ] 每页是否有明确的举措
- [ ] 左侧内容是否与右侧图示对应

### 6.3 设计层面

- [ ] 是否使用了专业图标（而非形状拼凑）
- [ ] 是否使用了专业图示（而非简单方块）
- [ ] 是否保持了风格统一

## 七、素材智能匹配规则

### 7.1 自动匹配逻辑

```python
def 匹配图标(内容关键词):
    匹配规则 = {
        "用户|客户|营业员": "人物图标",
        "系统|平台|CRM": "服务器图标",
        "流程|步骤|环节": "流程图标",
        "目标|指标|KPI": "目标图标",
        "数据|分析|统计": "图表图标",
        "问题|故障|诊断": "警告图标",
        "价值|收益|效果": "价值图标",
        "创新|亮点|特色": "星标图标"
    }
    return 匹配规则.get(内容关键词, "通用图标")
```

### 7.2 图模板自动推荐

```python
def 推荐图模板(内容类型, 页面类型):
    if 内容类型 == "流程":
        return "横向流程图"
    elif 内容类型 == "架构":
        return "分层架构图"
    elif 内容类型 == "指标":
        return "仪表盘图"
    elif 内容类型 == "价值":
        return "价值链图"
    else:
        return "内容卡片"
```
