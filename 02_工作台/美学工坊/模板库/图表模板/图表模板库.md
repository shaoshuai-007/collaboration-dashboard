## 图表选择指南

## 模板1：对比柱状图

### 适用场景

- 多维度数据对比
- 时间点对比
- 分类对比

### 设计规范

```
┌─────────────────────────────────────────┐
│            [图表标题：结论式]            │
├─────────────────────────────────────────┤
│                                         │
│  ████  35%                              │
│  ████                                   │
│  ████  ████  28%                        │
│  ████  ████  ████  22%                  │
│  ████  ████  ████  ████  15%            │
│                                         │
│  ───┬───┬───┬───┬───                    │
│     A   B   C   D                       │
│                                         │
└─────────────────────────────────────────┘
```

### 设计要点

| 要素 | 规范 |
|------|------|
| **标题** | 结论式标题，如"A类产品占比最高达35%" |
| **颜色** | 主要数据用品牌色，其他用灰色系 |
| **数据标签** | 在柱顶显示具体数值 |
| **排序** | 按数值大小排序（降序） |
| **间距** | 柱间距=柱宽的50% |

### 配色建议

| 情况 | 配色方案 |
|------|---------|
| 单一系列 | 渐变色（深→浅） |
| 多系列对比 | 对比色（蓝vs橙） |
| 正负对比 | 绿vs红 |

## 模板2：趋势折线图

### 适用场景

- 时间序列分析
- 趋势变化展示
- 预测分析

### 设计规范

```
┌─────────────────────────────────────────┐
│            [图表标题：趋势结论]          │
├─────────────────────────────────────────┤
│                                    ●●●  │
│                               ●●●●      │
│                          ●●●●           │
│                     ●●●●                │
│                ●●●●                     │
│           ●●●●                          │
│      ●●●●                               │
│ ●●●●                                    │
│                                         │
│  ─┬─────┬─────┬─────┬─────┬─           │
│   1月   2月   3月   4月   5月           │
│                                         │
│   [图例] ━ 实际值  ━ 目标值             │
└─────────────────────────────────────────┘
```

### 设计要点

| 要素 | 规范 |
|------|------|
| **标题** | 趋势结论，如"销售额持续增长，5月创新高" |
| **线条** | 粗细适中（2-3px），颜色区分 |
| **数据点** | 显示关键数据点标签 |
| **趋势线** | 可添加趋势线辅助判断 |
| **网格线** | 淡色、虚线，不喧宾夺主 |

### 多线对比

| 线条数 | 建议 |
|:------:|------|
| 1条 | 主色，加粗 |
| 2条 | 主色+辅助色 |
| 3-4条 | 不同颜色区分 |
| >4条 | 不推荐，考虑分组 |

## 模板3：占比环形图

### 适用场景

- 比例分析
- 构成展示
- 目标达成

### 设计规范

```
┌─────────────────────────────────────────┐
│            [图表标题：占比结论]          │
├─────────────────────────────────────────┤
│                                         │
│              ┌───────┐                  │
│           ╱─│       │─╲                │
│          │  │  45%  │  │  A类          │
│          │  │       │  │               │
│           ╲─│───────│─╱                │
│              └───────┘                  │
│                                         │
│   ████ A类 45%                          │
│   ████ B类 30%                          │
│   ████ C类 15%                          │
│   ████ D类 10%                          │
│                                         │
└─────────────────────────────────────────┘
```

### 设计要点

| 要素 | 规范 |
|------|------|
| **标题** | 占比结论，如"A类占比最高达45%" |
| **分段数** | 不超过5-6段 |
| **中心** | 显示核心数据（总数/核心指标） |
| **颜色** | 主色→辅助色渐变 |
| **图例** | 右侧或下方，加百分比 |

### 特殊应用

| 应用场景 | 说明 |
|---------|------|
| 目标达成 | 环形显示完成进度 |
| 对比环形 | 多个环形并排对比 |

## 模板4：流程漏斗图

### 适用场景

- 销售漏斗
- 转化分析
- 流程优化

### 设计规范

```
┌─────────────────────────────────────────┐
│            [图表标题：转化分析]          │
├─────────────────────────────────────────┤
│                                         │
│  ████████████████████████████████████   │
│  │        访问量 10000        │  100%   │
│   ██████████████████████████████        │
│   │      注册 5000         │    50%    │
│    ██████████████████████████           │
│    │     活跃 2000      │      20%     │
│     ████████████████████                │
│     │   付费 500    │        5%        │
│      ████████████                       │
│      │ 复购 100 │          1%          │
│       ████████                          │
│                                         │
└─────────────────────────────────────────┘
```

### 设计要点

| 要素 | 规范 |
|------|------|
| **标题** | 转化分析结论 |
| **层级** | 不超过5-6层 |
| **颜色** | 从上到下渐变或对比色 |
| **数据** | 显示绝对值和转化率 |
| **分析** | 标注流失严重环节 |

### 分析要点

| 指标 | 说明 |
|------|------|
| 转化率 | 各层之间的转化比例 |
| 流失量 | 各层流失的绝对数量 |
| 流失率 | 各层流失的比例 |
| 瓶颈 | 流失最严重的环节 |

## 图表设计通用规范

### 标题写法

**结论式标题**（推荐）：
- ✅ "销售额增长20%，华东区贡献最大"
- ✅ "转化率仅5%，注册→活跃是瓶颈"
- ❌ "销售数据图"

### 配色原则

| 原则 | 说明 |
|------|------|
| **一致性** | 同一报告内配色统一 |
| **对比性** | 需要对比时用对比色 |
| **层次性** | 重要数据用亮色/主色 |
| **色盲友好** | 避免仅用红绿区分 |

### 数据标签

| 类型 | 标签方式 |
|------|---------|
| 柱状图 | 柱顶显示数值 |
| 折线图 | 关键点显示数值 |
| 饼图 | 显示百分比 |
| 表格 | 数值对齐 |

### 图表组合

**仪表盘布局**：
```
┌─────────────────────────────────────────┐
│  [核心指标大卡片]                        │
├───────────────┬─────────────────────────┤
│  [趋势图]     │  [占比图]               │
├───────────────┴─────────────────────────┤
│  [详细数据表格]                          │
└─────────────────────────────────────────┘
```

## 使用指南

### 图表设计流程

| 步骤 | 动作 | 说明 |
|:----:|------|------|
| 1 | 明确目的 | 要表达什么结论 |
| 2 | 选择图表 | 按数据类型选择 |
| 3 | 设计样式 | 配色、标签、标题 |
| 4 | 检查清晰 | 用户能一眼看懂吗 |

### 南乔心法

> 图表是数据的翻译器。
> 
> **好的图表让数据说话，让结论自现。**
> 
> **一图一结论，标题即答案。**
