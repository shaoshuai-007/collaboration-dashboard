## 1. 设计四大原则（CRAP）⭐⭐⭐

### 核心原理

| 原则 | 英文 | 说明 | 应用场景 |
|------|------|------|---------|
| **对比** | Contrast | 突出差异，制造焦点 | CTA按钮、重要信息 |
| **重复** | Repetition | 建立一致性，强化品牌 | 配色、字体、元素 |
| **对齐** | Alignment | 建立秩序，提升专业 | 排版、布局 |
| **亲密** | Proximity | 相关元素靠近 | 信息分组、卡片设计 |

### 实践要点

**对比（Contrast）**：
- 大小对比：标题 > 正文 1.5-2倍
- 颜色对比：主色 vs 辅助色
- 粗细对比：加粗 vs 常规
- 留白对比：重点周围多留白

**重复（Repetition）**：
- 配色重复：同系列用同一配色
- 字体重复：标题用同一字体
- 元素重复：图标风格统一
- 间距重复：统一边距和间距

**对齐（Alignment）**：
- 左对齐：符合阅读习惯
- 居中对齐：正式、标题
- 右对齐：时间、数字
- 使用参考线确保对齐

**亲密（Proximity）**：
- 相关内容靠近放置
- 不相关内容分开
- 组内间距 < 组间间距
- 形成清晰的视觉分组

### 南乔心法

> 对比创造焦点，重复建立认同，对齐传递专业，亲密促进理解。

## 2. 视觉层次法 ⭐⭐⭐

### 核心原理

通过大小、颜色、位置建立信息层级，引导用户视线。

### 层次结构

```
第一层：焦点（最吸引注意）
    ↓ 大、亮、上/左
第二层：次要信息（辅助理解）
    ↓ 中、较暗
第三层：背景信息（补充细节）
    ↓ 小、淡
```

### 实现技巧

| 技巧 | 方法 | 效果 |
|------|------|------|
| **大小** | 重要信息放大1.5-2倍 | 视觉主导 |
| **颜色** | 重要信息用主色/高亮色 | 吸引注意 |
| **位置** | 重要信息放上方/左侧 | 优先阅读 |
| **粗细** | 标题加粗，正文常规 | 层次分明 |
| **间距** | 重要信息周围多留白 | 突出重点 |

### 视觉权重计算

| 因素 | 权重 | 说明 |
|------|:----:|------|
| 大小 | 30% | 越大越显眼 |
| 颜色 | 25% | 越亮越显眼 |
| 位置 | 20% | 越上/左越显眼 |
| 对比度 | 15% | 对比越强越显眼 |
| 留白 | 10% | 周围留白越多越显眼 |

## 3. 留白艺术法 ⭐⭐⭐

### 核心原理

留白不是浪费，而是呼吸的空间。

### 留白作用

| 作用 | 说明 |
|------|------|
| 减少视觉负担 | 信息不拥挤，阅读更轻松 |
| 突出核心内容 | 留白让重点更突出 |
| 提升高级感 | 高端设计往往留白更多 |
| 引导视觉流向 | 留白创造视觉路径 |

### 黄金比例

- 内容区 : 留白区 = 60 : 40
- 段落间距 ≥ 行高
- 元素间距 ≥ 元素本身

### 留白类型

| 类型 | 位置 | 作用 |
|------|------|------|
| 宏观留白 | 页面边缘、大区块之间 | 整体呼吸感 |
| 微观留白 | 字符、行、段落之间 | 阅读舒适度 |
| 感知留白 | 颜色对比、透明度 | 视觉轻松感 |

### 南乔心法

> 留白如水墨画之虚空，虽无笔墨，却意境深远。
> 密不透风处，需疏可走马；疏可走马处，需密不透风。

## 4. 对比统一法 ⭐⭐

### 核心原理

对比创造视觉冲击，统一保持整体和谐。好的设计在对比与统一之间找到平衡。

### 对比类型

| 对比类型 | 说明 | 效果 |
|---------|------|------|
| 大小对比 | 大vs小 | 主次分明 |
| 颜色对比 | 暖vs冷、亮vs暗 | 视觉冲击 |
| 形状对比 | 方vs圆、直vs曲 | 趣味性 |
| 粗细对比 | 粗vs细 | 层次感 |
| 疏密对比 | 密vs疏 | 节奏感 |

### 统一要素

| 要素 | 统一方式 |
|------|---------|
| 配色 | 使用同一色系 |
| 字体 | 标题/正文各统一一种 |
| 风格 | 图标、按钮风格统一 |
| 间距 | 统一网格系统 |

### 平衡原则

- 70% 统一 + 30% 对比 = 和谐有趣
- 过度对比 = 杂乱无章
- 过度统一 = 单调乏味

## 5. 格式塔原理 ⭐⭐

### 核心原理

人脑倾向于将视觉元素组织成有意义的整体。

### 五大原则

| 原则 | 说明 | 应用 |
|------|------|------|
| **接近性** | 距离近的元素被视为一组 | 信息分组 |
| **相似性** | 相似的元素被视为一类 | 分类设计 |
| **连续性** | 视线沿路径流动 | 视觉引导 |
| **闭合性** | 大脑自动补全不完整形状 | 图标设计 |
| **图形-背景** | 区分主体和背景 | 突出重点 |

### 设计应用

**接近性**：
- 相关功能放一起
- 导航分组
- 表单分组

**相似性**：
- 同类按钮用同一样式
- 同级标题用同一格式
- 同类信息用同一颜色
