## 1. 色彩心理学 ⭐⭐⭐

### 核心原理

不同颜色引发不同的情感联想，选择合适的颜色传递正确的情感。

### 色彩情感表

| 颜色 | 情感联想 | 正面意义 | 负面意义 | 适用场景 |
|------|---------|---------|---------|---------|
| 🔴 红色 | 热情、能量、警示 | 激情、力量、爱 | 危险、愤怒、冲动 | 促销、重要提醒、餐饮 |
| 🟠 橙色 | 活力、温暖、创意 | 友好、热情、创新 | 廉价、幼稚 | 创新产品、年轻品牌、运动 |
| 🟡 黄色 | 阳光、乐观、警示 | 快乐、希望、智慧 | 警告、胆怯、欺骗 | 儿童产品、提示信息、食品 |
| 🟢 绿色 | 自然、健康、成长 | 生命、和平、安全 | 嫉妒、幼稚 | 健康产品、环保主题、金融 |
| 🔵 蓝色 | 专业、信任、科技 | 稳定、智慧、忠诚 | 冷漠、忧郁 | 企业官网、科技产品、金融 |
| 🟣 紫色 | 优雅、神秘、智慧 | 高贵、创造、神秘 | 傲慢、哀伤 | 高端产品、创意设计、美容 |
| ⚪ 白色 | 纯净、简约、现代 | 纯洁、简约、清新 | 空虚、冷漠 | 极简设计、医疗健康、科技 |
| ⚫ 黑色 | 高端、神秘、权威 | 高贵、力量、优雅 | 死亡、恐惧、悲伤 | 奢侈品、高端品牌、时尚 |

### 色彩选择指南

**按情感选择**：
- 想传递"专业"→ 蓝色
- 想传递"活力"→ 橙色
- 想传递"高端"→ 黑色/紫色
- 想传递"健康"→ 绿色

**按行业选择**：
- 科技行业 → 蓝色、紫色
- 金融行业 → 蓝色、绿色
- 医疗健康 → 白色、绿色、蓝色
- 餐饮食品 → 红色、橙色、黄色
- 教育培训 → 蓝色、橙色

## 2. 60-30-10 配色法则 ⭐⭐⭐

### 核心原理

配色比例决定视觉和谐度。经典的黄金配色比例：60%主色+30%辅助色+10%点缀色。

### 法则详解

```
┌─────────────────────────────────────┐
│                                     │
│  60% 主色调（背景色）               │
│  - 建立整体基调                     │
│  - 通常中性色或浅色                 │
│  - 定义页面氛围                     │
│                                     │
│  30% 辅助色（次要素）               │
│  - 支撑主色调                       │
│  - 用于卡片、区块、导航等           │
│  - 与主色调和谐共存                 │
│                                     │
│  10% 点缀色（强调色）               │
│  - 吸引注意力                       │
│  - 用于按钮、链接、高亮等           │
│  - 与主色形成对比                   │
│                                     │
└─────────────────────────────────────┘
```

### 配色示例

**科技专业风**：
- 60% 深蓝 `#1a1a2e` - 沉稳背景
- 30% 中蓝 `#16213e` - 卡片区块
- 10% 金色 `#FFD700` - 点缀强调

**清新自然风**：
- 60% 浅绿 `#E8F5E9` - 清新背景
- 30% 深绿 `#2E7D32` - 主要元素
- 10% 橙色 `#FF9800` - 点缀强调

**高端商务风**：
- 60% 浅灰 `#F5F5F5` - 简洁背景
- 30% 深灰 `#333333` - 文字元素
- 10% 品牌色 - 点缀强调

### 注意事项

- ❌ 三种颜色面积相近 = 视觉混乱
- ❌ 点缀色面积过大 = 失去重点
- ✅ 主色占据主导 = 视觉统一

## 3. 经典配色方案 ⭐⭐⭐

### 方案总览

| 方案 | 构成 | 特点 | 适用场景 |
|------|------|------|---------|
| **单色配色** | 主色+明度变化 | 和谐统一、专业稳重 | 企业设计、专业文档 |
| **邻近色配色** | 色环相邻3色 | 自然舒适、柔和和谐 | 自然主题、健康产品 |
| **对比色配色** | 色环相对2色 | 强烈冲击、吸引注意 | 促销设计、强调重点 |
| **三角配色** | 色环等距3色 | 平衡丰富、活力创意 | 活力品牌、创意设计 |

### 详细说明

#### 单色配色

```
主色：#0066CC
├── 浅色：#4D94FF（+明度30%）
├── 中色：#0066CC（主色）
└── 深色：#003D7A（-明度30%）
```

**特点**：最安全、最和谐
**适用**：专业、稳重的设计

---

#### 邻近色配色

```
色环上相邻的3个颜色（色相差30-60°）

示例：蓝 → 蓝绿 → 绿
蓝 #0066CC → 蓝绿 #00AA88 → 绿 #00CC66
```

**特点**：自然过渡、舒适柔和
**适用**：自然、健康、成长主题

---

#### 对比色配色

```
色环上相对的两个颜色（色相差180°）

示例：蓝 ⟷ 橙
蓝 #0066CC ⟷ 橙 #FF9933
```

**特点**：强烈对比、视觉冲击
**适用**：需要突出、吸引注意的设计

**使用技巧**：
- 避免两种颜色面积相等
- 一主一辅，如80%蓝+20%橙

---

#### 三角配色

```
色环上等距的3个颜色（色相差120°）

示例：蓝 → 黄 → 红
蓝 #0066CC → 黄 #FFCC00 → 红 #FF3366
```

**特点**：平衡、丰富、活力
**适用**：年轻、活力、创意设计

**使用技巧**：
- 一主两次，如60%蓝+25%黄+15%红
- 降低饱和度更和谐

## 4. 行业配色规范 ⭐⭐

### 科技行业

| 风格 | 配色 | 情感 |
|------|------|------|
| **专业科技** | 蓝+灰+白 | 信任、专业 |
| **创新科技** | 蓝+绿+金 | 成长、创新 |
| **AI智能** | 紫+蓝+金 | 智慧、未来 |
| **数据可视化** | 蓝+绿+橙+红 | 多维、动态 |

**常用色值**：
- 科技蓝：`#0066CC`
- 深空蓝：`#0A1929`
- 亮蓝：`#4FC3F7`

---

### 金融行业

| 风格 | 配色 | 情感 |
|------|------|------|
| **稳健金融** | 蓝+金+白 | 信任、稳健 |
| **创新金融** | 蓝+绿+橙 | 创新、活力 |
| **高端金融** | 黑+金+白 | 高端、尊贵 |

**常用色值**：
- 金融蓝：`#1E3A8A`
- 金融金：`#FFD700`
- 稳健绿：`#10B981`

---

### 医疗健康

| 风格 | 配色 | 情感 |
|------|------|------|
| **专业医疗** | 蓝+白+绿 | 专业、洁净 |
| **健康生活** | 绿+白+橙 | 健康、活力 |
| **温和关怀** | 蓝+粉+白 | 温暖、关怀 |

**常用色值**：
- 医疗蓝：`#0EA5E9`
- 健康绿：`#22C55E`
- 温暖粉：`#F472B6`
