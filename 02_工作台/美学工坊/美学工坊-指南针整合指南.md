## 三、方案编写全流程整合

### 方案编写七步法（指南针+美学工坊）

```
第1步：需求理解
├── 指南针：明确目标、受众、场景
└── 美学工坊：确定行业、风格、配色

第2步：内容规划
├── 指南针：按模板确定章节结构
└── 美学工坊：选择合适的模板

第3步：结构设计
├── 指南针：应用金字塔原理
└── 美学工坊：PPT结构设计、页面规划

第4步：内容生成
├── 指南针：填充核心内容
└── 美学工坊：图表设计、图标选用

第5步：美化优化
├── 指南针：确保内容准确
└── 美学工坊：应用CRAP、配色、字体

第6步：质量检查
├── 指南针：逻辑检查、内容检查
└── 美学工坊：视觉检查、风格检查

第7步：输出交付
├── 指南针：按需求导出
└── 美学工坊：格式检查、最终优化
```

## 五、字体使用快速指南

### PPT字体规范

| 元素 | 字体 | 字号 |
|------|------|:----:|
| 封面标题 | 微软雅黑/思源黑体 | 44-60pt |
| 章节标题 | 微软雅黑 | 36-44pt |
| 页面标题 | 微软雅黑 | 28-36pt |
| 正文内容 | 微软雅黑 | 18-24pt |
| 注释说明 | 微软雅黑 | 12-16pt |

### 安全搭配

- **经典组合**：微软雅黑 + Arial
- **现代风格**：思源黑体 + Roboto
- **苹果风格**：苹方 + San Francisco

## 六、图表使用快速指南

### 数据展示选图指南

| 数据类型 | 推荐图表 | 说明 |
|---------|---------|------|
| 对比数据 | 柱状图 | 不同类别数值对比 |
| 趋势数据 | 折线图 | 时间序列变化趋势 |
| 占比数据 | 环形图/饼图 | 整体与部分关系 |
| 流程数据 | 漏斗图 | 流程转化分析 |
| 矩阵数据 | 散点图/矩阵图 | 多维度对比分析 |

### 图表设计要点

- 标题清晰：标题即结论
- 数据标签：关键数据标注
- 配色协调：与整体配色一致
- 避免装饰：去除冗余元素
