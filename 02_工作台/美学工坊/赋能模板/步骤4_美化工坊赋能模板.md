## 一、赋能定位

### 1.1 赋能目标

为指南针工程步骤4（方案PPT）提供三层赋能：
- **视觉化赋能**：让PPT更好看
- **结构化赋能**：让PPT更有条理
- **专业化赋能**：让PPT更专业

### 1.2 赋能关系

```
指南针工程步骤4（方案PPT）
        ↓
【美化工坊三层赋能】
    ├─ 视觉化赋能：配色、图标、字体、布局
    ├─ 结构化赋能：内容框架、信息架构、逻辑呈现
    └─ 专业化赋能：质量标准、检查清单、最佳实践
        ↓
高质量PPT模板 + 规范 + 标准
```

## 二、视觉化赋能

### 2.1 配色规范

#### 配色体系

| 角色 | 颜色名称 | RGB值 | HEX值 | 用途 |
|------|---------|-------|-------|------|
| 主色 | 钢蓝色 | RGB(70, 130, 180) | #4682B4 | 主要图形、标题装饰 |
| 主色深 | 深蓝色 | RGB(30, 60, 114) | #1E3C72 | 封面渐变、深色背景 |
| 点缀色 | 珊瑚红 | RGB(255, 107, 107) | #FF6B6B | 重点强调、第一顺位 |
| 辅色 | 矢车菊蓝 | RGB(100, 149, 237) | #6495ED | 辅助图形、第三顺位 |
| 中性色 | 深灰 | RGB(64, 64, 64) | #404040 | 正文文字 |
| 中性色 | 中灰 | RGB(128, 128, 128) | #808080 | 辅助文字 |
| 中性色 | 浅灰 | RGB(245, 247, 250) | #F5F7FA | 页面背景 |
| 中性色 | 白色 | RGB(255, 255, 255) | #FFFFFF | 卡片背景、文字反白 |

#### 配色黄金法则（60-30-10）

| 角色 | 占比 | 颜色 |
|------|:----:|------|
| 主色 | 60% | 钢蓝色、深蓝色 |
| 辅色 | 30% | 矢车菊蓝、中灰色 |
| 点缀色 | 10% | 珊瑚红 |

#### 顺位配色规则

- **第一顺位**：珊瑚红 - 用于最重要的内容
- **第二顺位**：钢蓝色 - 用于次重要的内容
- **第三顺位**：矢车菊蓝 - 用于辅助内容

### 2.2 图标尺寸规范

#### 标准尺寸（5级）

| 级别 | 宽度（英寸） | 高度（英寸） | 用途 |
|------|------------|------------|------|
| 特大 | 3.1 | 0.8 | 重点内容、主要模块 |
| 大 | 3.1 | 0.6-0.7 | 重要内容、业务场景 |
| 中 | 2.0 | 0.75 | 常规内容、标准图标 |
| 小 | 1.5 | 0.6-0.65 | 辅助内容、次要信息 |
| 超小 | 1.0 | 0.55 | 密集内容、补充信息 |

#### 页面类型与图标尺寸映射

| 页面类型 | 图标尺寸 | 说明 |
|---------|---------|------|
| 问题分析页 | 中（2.0×0.75） | 问题表象、根因、解决方案统一尺寸 |
| 能力页-业务场景 | 大（3.1×0.6） | 业务场景和核心能力一致 |
| 能力页-技术流程 | 中（2.0×0.75） | 技术流程步骤 |
| 系统集成页 | 自定义 | API网关、上游系统、下游系统不同尺寸 |

#### 图标尺寸一致性原则

**原则1**：同类型图标尺寸必须一致
- 同一页面内，相同类型的图标尺寸必须一致

**原则2**：层级关系清晰
- 上层内容图标 > 下层内容图标
- 主要内容图标 > 辅助内容图标

**原则3**：视觉平衡
- 每行图标总宽度应接近页面宽度
- 避免图标过大或过小导致视觉失衡

### 2.4 布局规范

#### 页面边距

| 区域 | 边距（英寸） |
|------|------------|
| 左边距 | 0.5-0.8 |
| 右边距 | 0.5-0.8 |
| 上边距 | 0.4-0.6 |
| 下边距 | 0.4-0.6 |

#### 内容间距

| 类型 | 间距（英寸） |
|------|------------|
| 标题与正文 | 0.4 |
| 段落之间 | 0.2 |
| 图标之间（横向） | 0.15-0.2 |
| 图标之间（纵向） | 0.1-0.15 |
| 图标与描述文字 | 0.05-0.1 |

#### 对齐规范

**水平对齐**：
- 标题行：左对齐，从0.85英寸开始
- 正文内容：左对齐，从0.8英寸开始
- 图标组：居中对齐或左对齐
- 单个居中元素：水平居中于页面

**垂直对齐**：
- 图标内的文字：垂直居中（MSO_ANCHOR.MIDDLE）
- 图标组：顶部对齐
- 多行内容：顶部对齐

#### 布局模式（4种）

**模式A：左右分栏（常用）**
```
| 左侧（40%）| 右侧（60%）|
| 重点举措   | 图示/数据  |
```

**模式B：上下分层**
```
| 标题区      |
| 内容区      |
| 图表区      |
```

**模式C：网格式**
```
| 图标1 | 图标2 | 图标3 |
| 图标4 | 图标5 | 图标6 |
```

**模式D：流式**
```
| 图标1 |
   ↓
| 图标2 |
   ↓
| 图标3 |
```

### 2.5 视觉效果规范

#### 阴影效果

**参数设置**：
- 模糊半径：50800 EMU（约0.05英寸）
- 距离：38100 EMU（约0.04英寸）
- 方向：2700000（向下）
- 透明度：40%

**应用场景**：
- 所有圆角矩形图标
- 所有卡片式元素
- 不应用于：装饰线、纯文字

#### 渐变效果

**线性渐变**：
- 角度：90度（从上到下）
- 停止点：2个
- 颜色1：主色深
- 颜色2：主色

**应用场景**：
- 封面背景
- 结束页背景

#### 圆角效果

**圆角矩形**：
- 使用MSO_SHAPE.ROUNDED_RECTANGLE
- 自动圆角，无需手动设置

## 三、结构化赋能

### 3.1 内容框架标准（三要素）

**每页必含三要素**：

| 要素 | 定义 | 格式 | 要求 |
|------|------|------|------|
| 标题 | 页面核心主题 | 主标题：副标题 | 副标题不超过10字 |
| 建设目标 | 页面核心目标 | 一句话说明 | 清晰、明确、可衡量 |
| 具体举措 | 具体行动 | 3-5条 | 每条包含标题+内容 |

---

### 3.2 内容扩充标准

#### 问题分析页

| 内容 | 要求 |
|------|------|
| 问题表象 | ≥50字 |
| 核心根因 | ≥6项，每项"概括+描述" |
| 解决方案 | ≥6项，每项"概括+描述" |
| 解决方案设计 | ≥150字，分段排列 |

#### 架构页

| 内容 | 要求 |
|------|------|
| 每层功能 | ≥50字 |
| 每层组成 | 3-5项，每项"名称+描述" |

#### 能力页

| 内容 | 要求 |
|------|------|
| 核心业务场景 | 3-4个，每个"场景名称+场景描述" |
| 技术实现路径 | 5-6步 |
| 关键指标 | 3-4个 |
| 业务描述 | ≥100字 |

---

### 3.3 信息架构方法论

#### 金字塔原理

**核心原则**：
1. 结论先行：每页先说结论
2. 以上统下：上层观点统领下层
3. 归类分组：同类信息归类呈现
4. 逻辑递进：按逻辑顺序组织

**结构示例**：
```
                    结论
                     │
         ┌───────────┼───────────┐
         │           │           │
       论点1       论点2       论点3
         │           │           │
    ┌────┼────┐ ┌────┼────┐ ┌────┼────┐
   论据 论据 论据 论据 论据 论据 论据 论据
```

#### SCQA开场设计

**结构**：
```
S（情境）：描述当前稳定状态
C（冲突）：描述打破稳定的事件
Q（问题）：提出需要解决的关键问题
A（答案）：给出解决方案核心观点
```

**示例**：
```
S：湖北电信持续推进数字化转型，业务运营效率稳步提升。
C：但在AI智能配案领域，竞争对手已率先布局，市场份额受到威胁。
Q：如何在短期内实现AI智能配案突破，抢占市场先机？
A：通过"1+3+N"智能配案体系，快速构建差异化竞争优势。
```

#### MECE原则

**核心要求**：
- 无遗漏（Mutually Exclusive）：分类完整，没有遗漏
- 无重叠（Collectively Exhaustive）：分类清晰，没有重叠

## 五、赋能应用流程

### 5.1 自动化赋能机制

```
指南针工程步骤4触发
        ↓
调用美化工坊赋能模板
        ↓
应用三层赋能
    ├─ 视觉化赋能：配色、图标、字体、布局
    ├─ 结构化赋能：内容框架、信息架构、逻辑呈现
    └─ 专业化赋能：质量标准、检查清单、最佳实践
        ↓
生成高质量PPT内容
        ↓
执行质量检查
        ↓
输出最终PPT
```

### 5.2 南乔执行标准

当执行指南针工程步骤4时，南乔必须：

1. **自动调用美化工坊赋能模板**
2. **应用视觉化赋能**：
   - 选择配色方案（钢蓝+珊瑚红）
   - 确定图标尺寸（根据页面类型）
   - 应用字体规范
   - 应用布局规范

3. **应用结构化赋能**：
   - 应用三要素标准
   - 应用内容扩充标准
   - 应用信息架构方法论

4. **应用专业化赋能**：
   - 执行质量检查
   - 应用最佳实践
   - 生成检查报告
