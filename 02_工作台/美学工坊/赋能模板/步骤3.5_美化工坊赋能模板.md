## 二、赋能三层架构

### 2.1 视觉化赋能

| 赋能项 | 内容 | 实现方式 |
|--------|------|----------|
| **文档排版优化** | 应用文档排版七要素 | 字体、行距、段落、标题规范 |
| **成本收益分析图美化** | 配色协调、对比清晰 | 钢蓝+珊瑚红配色 |
| **风险评估矩阵美化** | 四象限清晰、颜色醒目 | 专业图表设计 |
| **数据可视化** | 图表清晰、一目了然 | 专业图表设计 |

### 2.2 结构化赋能

| 赋能项 | 内容 | 实现方式 |
|--------|------|----------|
| **可行性框架标准化** | 技术+经济+运营+时间可行性 | 四维度分析 |
| **成本收益分析完整** | 成本项+收益项+ROI计算 | 三要素完整 |
| **风险评估完整** | 概率+影响+应对措施 | 三项齐全 |
| **结论明确** | 可行/不可行/有条件可行 | 结论清晰 |

### 2.3 专业化赋能

| 赋能项 | 内容 | 实现方式 |
|--------|------|----------|
| **质量检查** | 内容质量+逻辑质量 | 检查清单 |
| **方法论正确性** | 成本收益分析、风险评估矩阵 | 方法论检查 |
| **最佳实践** | 可行性报告模板、检查清单 | 模板复用 |

## 三、自动化赋能流程

```
步骤3.5完成 → 触发赋能
    ↓
读取可行性验证数据（MD格式）
    ↓
【视觉化赋能】
    ├─ 应用文档排版规范
    ├─ 生成成本收益分析图（PNG格式）
    ├─ 生成风险评估矩阵（PNG格式）
    └─ 生成投资回报测算表（Excel格式）
    ↓
【结构化赋能】
    ├─ 应用可行性报告模板
    ├─ 检查四维度分析（技术、经济、运营、时间）
    ├─ 检查成本收益分析完整性
    ├─ 检查风险评估三项齐全
    └─ 检查结论明确性
    ↓
【专业化赋能】
    ├─ 质量检查（内容质量40分+逻辑质量20分）
    ├─ 方法论正确性检查
    ├─ 生成检查清单
    └─ 提供改进建议
    ↓
输出Word格式报告 + PNG格式图片 + Excel格式表格
```

## 四、视觉规范应用

### 4.1 配色规范

| 元素 | 颜色 | 用途 |
|------|------|------|
| **主色** | 钢蓝 RGB(70,130,180) | 主要内容、标题 |
| **辅色** | 珊瑚红 RGB(255,107,107) | 重点突出、风险警示 |
| **中性色** | 浅灰 RGB(245,247,250) | 背景、分隔 |

### 4.2 成本收益分析图设计规范

```
┌────────────────────────────────────────┐
│  成本收益分析                           │
├────────────────────────────────────────┤
│                                        │
│  成本项                收益项           │
│  ┌──────────┐         ┌──────────┐    │

│  │  ¥50万   │         │  ¥80万   │    │
│  └──────────┘         └──────────┘    │
│                                        │
│  ┌──────────┐         ┌──────────┐    │
│  │ 技术成本  │         │ 成本节约  │    │
│  │  ¥30万   │         │  ¥60万   │    │
│  └──────────┘         └──────────┘    │
│                                        │
│  ┌──────────┐         ┌──────────┐    │
│  │ 运营成本  │         │ 收入增长  │    │
```

### 4.3 风险评估矩阵设计规范

```
┌────────────────────────────────────────┐
│  风险评估矩阵                           │
├────────────────────────────────────────┤
│  高影响                                 │
│    ↑                                   │
│    │  高风险区                          │
│    │  ┌──┐  ┌──┐                       │
│    │  │R1│  │R2│                       │
│    │  └──┘  └──┘                       │
│    │                                   │
│────┼───────────────────→ 高概率         │
│    │                                   │
│    │  低风险区                          │
│    │  ┌──┐  ┌──┐                       │
│    │  │R3│  │R4│                       │
│    │  └──┘  └──┘                       │
│    ↓                                   │
│  低影响                                 │
│        低概率                           │
└────────────────────────────────────────┘
```

**颜色编码**：
- 🔴 高风险区（高影响+高概率）：珊瑚红
- 🟠 中风险区（高影响+低概率 或 低影响+高概率）：橙色
- 🟢 低风险区（低影响+低概率）：绿色
