# PPT改进技能 - 使用美学工坊模板

## 问题诊断

| 问题 | 当前状态 | 改进方案 |
|------|---------|---------|
| 无故事主线 | 只有信息堆砌 | 每页按"故事-目标-举措"设计 |
| 图示选择随意 | 随机选择 | 按业务逻辑匹配图示类型 |
| 模板未生效 | 用Python脚本生成 | 基于美学工坊模板规范设计 |
| 素材未利用 | 无图标/组件 | 集成素材库（500+图标） |

## 改进步骤

### 步骤1：使用模板规范

每页PPT包含：
- **故事**：这页要讲什么故事？
- **目标**：想让读者理解什么？
- **举措**：如何实现？
- **图示**：基于业务逻辑选择（见下表）

### 步骤2：图示类型选择表

| 业务逻辑 | 图示类型 | 示例 |
|---------|---------|------|
| 线性流程 | 横向流程图 | 用户提问→意图识别→知识检索 |
| 分层架构 | 六层架构图 | 交互层→智能体层→能力层→模型层→数据层→集成层 |
| 时间序列 | 时间轴图 | Q1→Q2→Q3→Q4 |
| 指标对比 | 仪表盘/卡片 | 现状30分钟→目标5分钟 |
| 问题分析 | 冰山模型 | 表象→根因→解决方案 |
| 多系统 | 星型拓扑图 | 中心节点+边缘节点 |
| 决策分支 | 决策树 | 不同问题的诊断路径 |
| 价值递进 | 价值链图 | 考核达标→效率提升→体验改善 |

### 步骤3：素材集成

```python
# 使用美学工坊素材库
素材库路径 = "/root/.openclaw/workspace/02_工作台/美学工坊/素材库/PPT素材库/"
图标库 = 素材库路径 + "02_图标库/"
架构图组件 = 素材库路径 + "03_架构图组件/"
流程图组件 = 素材库路径 + "04_流程图组件/"
```

## 生成流程

```
1. 读取内容大纲
2. 按模板规范设计每页：
   - 确定故事主线
   - 匹配图示类型
   - 选择配色方案
   - 集成图标组件
3. 输出美化PPT
```

## 质量检查

| 检查项 | 标准 | 分值 |
|--------|------|:----:|
| 故事主线 | 每页有明确故事 | 20 |
| 图示匹配 | 图示类型符合业务逻辑 | 20 |
| 配色方案 | 符合品牌色/商务色 | 15 |
| 布局一致 | 相似页面布局统一 | 15 |
| 素材使用 | 图标/组件美化页面 | 15 |
| 信息清晰 | 无重叠、无溢出 | 15 |
| **总分** | **≥90分** | **100** |
