## 一、问题诊断

### 当前问题

```

```

```

Run `openclaw configure --section web` to store it,
or set BRAVE_API_KEY in the Gateway environment.
```

---

## 二、解决方案

### 方案1：配置 Brave Search API（推荐）

#### 步骤1：获取 Brave Search API Key

1. 访问：https://brave.com/search/api/
2. 注册/登录 Brave 账户
3. 创建 API Key
4. 免费额度：2000次/月

#### 步骤2：配置到 OpenClaw

**方法A：交互式配置**
```bash
openclaw configure --section web
# 按提示输入 Brave API Key
```

**方法B：环境变量配置**
```bash
# 在 Gateway 环境变量中添加
export BRAVE_API_KEY="your_api_key_here"
```

**方法C：配置文件**
```bash
# 编辑 Gateway 配置文件
# 添加：
{
  "web": {
    "braveApiKey": "your_api_key_here"
  }
}
```

#### 步骤3：验证配置

```bash
# 重启 Gateway
openclaw gateway restart

# 测试搜索
# 在对话中让 AI 执行 web_search
```
