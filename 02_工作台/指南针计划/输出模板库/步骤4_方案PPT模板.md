## SCQA开场设计

### 模板

```
┌────────────────────────────────────────┐
│                                        │
│    S（情境）：[描述当前稳定状态]        │
│                                        │
│    C（冲突）：[描述打破稳定的事件]      │
│                                        │
│    Q（问题）：[提出需要解决的关键问题]  │
│                                        │
│    A（答案）：[给出解决方案核心观点]    │
│                                        │
└────────────────────────────────────────┘
```

### 示例

```
S：湖北电信持续推进数字化转型，业务运营效率稳步提升。

C：但在AI智能配案领域，竞争对手已率先布局，市场份额受到威胁。

Q：如何在短期内实现AI智能配案突破，抢占市场先机？

A：通过"1+3+N"智能配案体系，快速构建差异化竞争优势。
```

## 金字塔原理应用

### 核心原则

1. **结论先行**：每页PPT先说结论
2. **以上统下**：上层观点统领下层
3. **归类分组**：同类信息归类呈现
4. **逻辑递进**：按逻辑顺序组织

### 结构示例

```
                    结论
                     │
         ┌───────────┼───────────┐
         │           │           │
       论点1       论点2       论点3
         │           │           │
    ┌────┼────┐ ┌────┼────┐ ┌────┼────┐
   论据 论据 论据 论据 论据 论据 论据 论据
```
