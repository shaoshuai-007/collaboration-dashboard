## 模板使用规范

### 选择原则

1. **风格匹配**：根据主题选择合适风格
   - AI/科技主题 → 科技风
   - 企业报告 → 商务风
   - 创意展示 → 创意风

2. **受众分析**：根据受众选择
   - 高管汇报 → 简约商务风
   - 技术分享 → 科技风
   - 客户提案 → 专业商务风

3. **场景适配**：根据场景选择
   - 线上演示 → 深色背景更佳
   - 线下打印 → 浅色背景更佳

### 质量标准

| 指标 | 要求 |
|:----:|------|
| 美观度 | ⭐⭐⭐⭐⭐ 高大上 |
| 专业性 | 设计专业，排版规范 |
| 易读性 | 层次清晰，信息明确 |
| 一致性 | 风格统一，配色协调 |
| 可编辑性 | 易于修改，灵活调整 |
