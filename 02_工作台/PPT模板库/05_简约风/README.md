## 设计特点

简约风模板遵循"Less is More"原则：

1. **配色方案**
   - 单色系为主
   - 大量留白
   - 低饱和度

2. **设计元素**
   - 几何图形
   - 简洁线条
   - 最少装饰

3. **版式布局**
   - 清晰层次
   - 重点突出
   - 视觉平衡

---

## 简约设计原则

1. **内容优先**
   - 突出核心信息
   - 避免视觉干扰
   - 信息层次分明

2. **视觉呼吸**
   - 适当留白
   - 元素间距合理
   - 不拥挤

3. **一致性**
   - 统一字体
   - 统一配色
   - 统一风格

---

## 使用建议

1. **内容精简**
   - 一页一主题
   - 关键词代替句子
   - 图文结合

2. **字体选择**
   - 无衬线字体
   - 2-3种字号
   - 清晰易读

3. **配色控制**
   - 主色 + 辅助色
   - 不超过3种颜色
   - 保持统一
