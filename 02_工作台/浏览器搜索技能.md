# 浏览器搜索技能

> **更新时间**：2026-03-12
> **状态**：可用
> **替代方案**：使用 browser 工具进行网页搜索

## 一、当前搜索能力

### ✅ 可用方案：浏览器工具

南乔已有 `browser` 工具，可以通过浏览器访问搜索引擎获取信息！

### 使用方式

```
1. 使用 browser 工具打开搜索引擎
2. 输入搜索关键词
3. 抓取搜索结果页面
4. 提取需要的信息
```

### 支持的搜索引擎

| 搜索引擎 | 网址 | 特点 |
|---------|------|------|
| **百度** | https://www.baidu.com | 中文搜索优化 |
| **必应** | https://www.bing.com | 国际化搜索 |
| **Google** | https://www.google.com | 全球最大搜索引擎 |

## 二、浏览器搜索流程

### 标准流程

```
Step 1: 打开搜索引擎
browser action=open targetUrl="https://www.baidu.com"

Step 2: 输入搜索词
browser action=act request={kind: "type", ref: "搜索框", text: "搜索关键词"}

Step 3: 点击搜索
browser action=act request={kind: "click", ref: "搜索按钮"}

Step 4: 抓取结果
browser action=snapshot

Step 5: 提取信息
从快照中提取需要的信息
```

## 四、浏览器搜索 vs API搜索

| 维度 | 浏览器搜索 | API搜索 |
|------|-----------|---------|
| **速度** | 较慢（需加载页面） | 快（直接返回数据） |
| **准确性** | 高（真实搜索结果） | 取决于API质量 |
| **成本** | 免费 | 可能需要付费 |
| **限制** | 无明显限制 | API调用次数限制 |
| **可用性** | ✅ 立即可用 | ⏳ 需配置 |

|------|------|

| **批量查询** | 需要批量处理 |
| **自动化流程** | 需要高度自动化的场景 |
