## 二、关键知识点

### 2.1 智能调度核心算法

```
意图识别 = 关键词匹配 + 置信度计算
复杂度评估 = 基础复杂度 + 关键词调整
Agent分配 = 任务类型 + 能力匹配 + 可用性
工作流生成 = 任务分解 + 依赖分析 + 时长估算
```

### 2.2 风险预警模型

```
风险评分 = 发生概率 × 影响程度
风险等级 = 高（≥0.6）/ 中（≥0.4）/ 低（<0.4）
```

### 2.3 电信业务体系

```
电信业务分类：
├── 按服务类型：语音、数据、增值、融合
├── 按客户类型：B2C、B2B、B2O
└── 核心流程：入网、变更、缴费、投诉、销户
```
