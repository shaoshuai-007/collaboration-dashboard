# TOOLS.md

## 核心知识库检索工具
- **tool: 市场洞察检索**
  - 描述：查询最新的中国电信市场动态、AI应用趋势、运营商财报解读。
  - 数据源：`knowledge_base/01_市场洞察/《中国电信市场年度洞察与AI应用地图》.md`
- **tool: 竞品分析检索**
  - 描述：获取主要竞争对手（华为、亚信等）的产品、报价、策略及弱点。
  - 数据源：`knowledge_base/02_竞品分析/《竞品对标分析档案（打法库）》.md`
- **tool: 术语与场景解释**
  - 描述：解释电信行业术语，并将其与AI解决方案关联。
  - 数据源：`knowledge_base/03_术语词典/《电信行业黑话与业务场景词典》.md`
- **tool: 项目管理SOP查询**
  - 描述：提供营销项目全生命周期的标准操作流程，包括各阶段交付物和签批节点。
  - 数据源：`knowledge_base/04_SOP/《营销项目全生命周期SOP手册》.md`
- **tool: 目标进度看板查询**
  - 描述：查看当前的周/月度目标拆解、商机储备、回款预警等可视化数据。
  - 数据源：`knowledge_base/05_进度看板/《周_月度目标拆解与进度可视化看板》.md`
- **tool: 风险预警查询**
  - 描述：列出电信项目中常见风险及预设的应对预案。
  - 数据源：`knowledge_base/06_风险/《风险预警与风控清单》.md`
- **tool: 客户痛点案例检索**
  - 描述：查询过往与客户交流的一手痛点记录，用于支撑方案故事。
  - 数据源：`knowledge_base/07_客户痛点/《客户痛点访谈纪要库》.md`
- **tool: 解决方案模板检索**
  - 描述：获取针对特定电信营销场景（如5G升舱、存量维系）的标准AI赋能方案胶片。
  - 数据源：`knowledge_base/08_解决方案模板/《场景化解决方案胶片模板库》.md`
- **tool: 商务谈判策略检索**
  - 描述：提供价格谈判中的筹码交换策略和针对不同采购风格的话术建议。
  - 数据源：`knowledge_base/09_商务谈判/《商务谈判筹码与话术清单》.md`
- **tool: 复盘报告模板生成**
  - 描述：生成项目结项复盘报告的框架，引导用户填写目标、结果、失误、改进。
  - 数据源：`knowledge_base/10_复盘/《项目结项复盘报告（模板）》.md`
- **tool: 实战案例检索**
  - 描述：查询过往项目中的“杀虫剂案例”（失败教训与修复过程），帮助团队避坑。
  - 数据源：`knowledge_base/11_案例库/《一线实战案例库（杀虫剂案例）》.md`

## 外部工具（可选扩展）
- **tool: 日历管理**：对接企业日历，自动设置项目里程碑提醒。
- **tool: 邮件/IM 发送**：向指定人员发送项目周报或风险预警。
- **tool: 数据分析查询**：连接内部 BI 系统，查询实时的营销活动数据。
