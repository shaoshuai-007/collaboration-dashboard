# ⚡ 南乔执行检查清单

**用途**：每次执行任务时对照检查，确保不漏项

---

## ✅ 接收文件检查清单

**当少帅上传文件时**，按顺序检查：

```
□ 1. 文件存在吗？→ 检查路径
□ 2. 备份了吗？→ 知识库/原始文件/日期/
□ 3. 提取了吗？→ PDF/Word/Excel
□ 4. 分类了吗？→ 电信/AI/方法论/模板
□ 5. 创建知识文档了吗？→ MD格式
□ 6. 保存完整内容了吗？→ TXT格式
□ 7. 更新索引了吗？→ INDEX.md
□ 8. 分发团队了吗？→ 各Agent学习资料目录
□ 9. 完成汇报了吗？→ 向少帅汇报
```

**不能漏项！尤其是第8步！**

---

## ✅ 执行任务检查清单

**当需要做某件事时**，按顺序检查：

```
□ 1. 有现成技能吗？→ 查南乔技能快速索引.md
□ 2. 有 → 直接用
□ 3. 没有 → 再考虑其他方式
```

**不用现有工具 = 傻！**

---

## ✅ 搜索时效性检查清单

**当需要搜索时**，检查：

```
□ 1. 关键词包含"2026"或"最新"吗？
□ 2. 检查搜索结果的发布时间了吗？
□ 3. 确保获取最新内容了吗？
```

**不能用旧数据！**

---

## ✅ 完成任务后检查清单

**当完成重要任务后**，检查：

```
□ 1. 调用自我进化技能了吗？
□ 2. 生成反思报告了吗？
□ 3. 保存到进化日志了吗？
```

**有技能要用！**
