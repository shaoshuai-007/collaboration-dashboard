# ⚡ 南乔执行检查清单

**用途**：每次执行任务时对照检查，确保不漏项

---

## ✅ 接收文件检查清单

**当少帅上传文件时**，按顺序检查：

```
□ 1. 文件存在吗？→ 检查路径
□ 2. 备份了吗？→ 知识库/原始文件/日期/
□ 3. 提取了吗？→ PDF/Word/Excel
□ 4. 分类了吗？→ 电信/AI/方法论/模板
□ 5. 创建知识文档了吗？→ MD格式
□ 6. 保存完整内容了吗？→ TXT格式
□ 7. 更新索引了吗？→ INDEX.md
□ 8. 分发团队了吗？→ 各Agent学习资料目录
□ 9. 完成汇报了吗？→ 向少帅汇报
```

**不能漏项！尤其是第8步！**

---

## ✅ 执行任务检查清单

**当需要做某件事时**，按顺序检查：

```
□ 1. 有现成技能吗？→ 查南乔技能快速索引.md
□ 2. 有 → 直接用
□ 3. 没有 → 再考虑其他方式
```

**不用现有工具 = 傻！**

---

## ✅ 搜索时效性检查清单

**当需要搜索时**，检查：

```
□ 1. 关键词包含"2026"或"最新"吗？
□ 2. 检查搜索结果的发布时间了吗？
□ 3. 确保获取最新内容了吗？
```

**不能用旧数据！**

---

## ✅ 完成任务后检查清单

**当完成重要任务后**，检查：

```
□ 1. 调用自我进化技能了吗？
□ 2. 生成反思报告了吗？
□ 3. 保存到进化日志了吗？
```

**有技能要用！**

---

## ✅ 产出物审核检查清单

**当Agent生成产出物后**，执行：

```
□ 1. 调用闭环质量系统了吗？
   → python3 scripts/closed_loop_executor.py
   或
   → from scripts.closed_loop_executor import ClosedLoopExecutor
   
□ 2. 选择审核模式了吗？
   → smart：智能判断（推荐）
   → semi：半自动（重要文档）
   → auto：全自动（普通文档）

□ 3. 审核等级达到B级以上了吗？
   → A级（90-100分）：直接通过
   → B级（80-89分）：小幅优化后通过
   → C级（70-79分）：需要修复
   → D级（<70分）：需要重新编写

□ 4. 问题已修复并通过复审了吗？

□ 5. 审核报告保存了吗？
   → 03_输出成果/审核记录/
```

**双重审核闭环：自动审核 + AI修复 + 复审通过！**

**快速调用代码**：
```python
from scripts.closed_loop_executor import ClosedLoopExecutor

executor = ClosedLoopExecutor()
result = executor.execute("文档路径.md", mode="smart")

print(f"等级：{result['final_grade']}")  # 目标：A或B
print(f"得分：{result['final_score']}")  # 目标：≥80
```

## ✅ Agent调用检查清单

**当调用Agent技能时**，按顺序执行：

```
□ 1. 确定Agent和任务
   → Agent：采薇/织锦/呈彩/工尺/玉衡/筑台/折桂/扶摇/天工/知微
   → 任务：描述清楚要做什么
   → 输出：明确输出文件路径

□ 2. 调用Agent执行任务
   → 使用 sessions_spawn 或 skill 工具
   
□ 3. 自动触发审核（必做！）
   → from scripts.agent_wrapper import run_agent_with_audit
   → result = run_agent_with_audit(agent_name, task, output_file)

□ 4. 检查审核结果
   → result["audit_status"] == "passed" → 通过，通知少帅
   → result["audit_status"] != "passed" → 未通过，等待指示

□ 5. 审核未通过时
   → 诊断问题 → 修复 → 复审 → 循环直到通过
```

**快速调用代码**：
```python
from scripts.agent_wrapper import run_agent_with_audit

result = run_agent_with_audit(
    agent_name="采薇",
    task="生成湖北电信AI配案需求文档",
    output_file="03_输出成果/需求文档.md"
)

if result["audit_status"] == "passed":
    # ✅ 审核通过，发送给少帅
    message.send("✅ 审核通过...")
else:
    # ⚠️ 审核未通过，等待指示
    message.send("⚠️ 审核未通过...")
```

**Agent必须审核！这是铁律！**
