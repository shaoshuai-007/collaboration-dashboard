## 一、建设背景

### 1.1 集团要求

**《市场空间洞察工作指引1.0》**：
- 坚持以客户为中心，精准洞察客户需求
- 提升市场空间洞察能力

**集团对市场空间洞察系统构建工作要求**：
- 公众网格场景：客户、产品、渠道、物理空间和资源入库
- 存量经营场景：业务使用、消费行为和终端使用特征入库

| 问题 | 现状数据 |

### 2.2 两大能力

| 可视化洞察分析能力 | 全方位、快速地刻画各类业务客户分群及空间分群特征 |

### 2.3 六大举措

## 三、核心功能

### 3.1 客户群统一管理

| 功能 | 说明 |
|------|------|
| 3类客群纳管 | 用户级、客户级、异网级 |
| 业务分类清晰 | 提供业务分类口径，便于建立超细分人群 |
| 分群维度丰富 | 分群目标 + 分群用途打标 |
| 群生命周期管理 | 重新建、编辑到下线，完整管理 |

### 3.2 群分布可视化呈现

**群规模趋势**：
- 15天、30天、近60天变化趋势
- 一目了然

**地域分布**：
- 支持省-市-区县层级下钻
- 快速了解分群分布是否均匀

**网格分布**：
- 四级网格和5级网格分布
- 从网格角度了解群体分布

### 3.3 群空间可视化洞察

**多维度空间洞察**：
- 查看当前分群用户在各类业务上的渗透率
- 分析业务之间的潜在关联性

**多维度指标**：
- 智能音响、云盘黄金会员、云回看、千兆
- 小合约、全屋WiFi、FTTR、5G
- 橙分期等

### 3.4 群特征可视化分析

**业务特征图形化展示**：
- 套餐分布、移动终端分布
- 套餐流量分布、网龄分布

**极值自动计算**：
- 最大值、最小值、中位数

**画像特征立体化**：
- 年龄分布、性别、城乡分布
- 白领用户占比、K12家长占比、婴幼儿家长占比、有车一族占比

### 3.5 可视化群洞察一点配置

**流程**：
```
新建分群或选择已有分群 → 选择洞察指标 → 分群可视化洞察
```

## 五、对AI+客户群洞察项目的启示

### 5.1 现状问题复用

- 客户群有效占比低（4.1%）是普遍问题
- 人工取数周期长、效率低
- 客户群重复、冗余、复用率低

### 5.2 功能借鉴

| 功能 | 可借鉴点 |
|------|----------|
| 客户群统一管理 | 多维度分类、生命周期管理 |
| 可视化洞察 | 地域分布、网格分布、空间分析 |
| 一点配置 | 简化操作流程、快速生成 |

### 5.3 AI增强方向

1. **智能分群推荐**

   - 基于历史效果优化分群规则

2. **标签智能选择**
   - 解决6000+标签选择难问题
   - 自然语言→标签推荐

3. **分群效果预测**
   - 预测分群规模
   - 预测转化率
