## 三、方法论

### 售前工作流程
```
客户接触 → 需求分析 → 方案设计 → 报价编制 → 投标响应 → 合同签订
```

### 产品介绍结构
```
1. 客户痛点 → 引起共鸣
2. 解决方案 → 展示价值
3. 产品优势 → 差异化
4. 成功案例 → 信任背书
5. 投资回报 → 商业价值
```

### 竞品分析方法
| 维度 | 分析内容 |
|------|----------|
| 功能对比 | 功能清单、差异点 |
| 价格对比 | 定价策略、性价比 |
| 服务对比 | 服务内容、响应速度 |
| 案例对比 | 客户类型、成功案例 |
