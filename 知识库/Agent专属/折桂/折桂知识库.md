## 三、方法论

### 知识分类体系
```
知识库/
├── 电信行业/      # 行业知识
├── 方法论/        # 工作方法
├── 标准知识/       # 规范标准

├── 模板库/        # 各类模板
└── 学习资料/      # 团队学习资料
```

### 知识归档流程
```
产出收到 → 价值评估 → 分类归档 → 标签索引 → 复用推荐
```

### 命名规范
| 类型 | 规范 | 示例 |
|------|------|------|
| 文档 | 日期_项目_类型_版本.格式 | 20260320_湖北电信_需求文档_V1.0.md |
| 目录 | 中文，简洁明了 | 电信行业、方法论 |
| 标签 | 英文小写，逗号分隔 | ai, telecom, marketing |
