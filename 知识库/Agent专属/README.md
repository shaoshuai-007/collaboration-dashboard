## 知识库结构

每个Agent知识库包含：

```
1. 核心能力
   - 专业技能
   - 方法论

2. 常用模板
   - 模板名称
   - 路径
   - 用途

3. 方法论
   - 工作流程
   - 设计规范
   - 质量标准

4. 知识来源
   - 学习资料
   - 来源出处

5. 协作关系
   - 与其他Agent的协作
```

## 使用方式

### 1. Agent启动时
```
读取专属知识库 → 加载核心能力 → 准备工作
```

### 2. 任务执行时
```
读取相关知识模板 → 按模板执行 → 输出结果
```

### 3. 学习提升时
```
读取学习资料 → 学习新知识 → 更新知识库
```
