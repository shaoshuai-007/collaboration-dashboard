### 3.2 预警规则

```python
def predict_churn_risk(user):
    """
    预测用户流失风险
    
    风险等级：
    - 高风险：得分≥70
    - 中风险：得分40-69
    - 低风险：得分<40
    """
    score = 0
    
    # ARPU下降
    if user.arpu_decline >= 0.2:
        score += 30
    
    # 流量下降
    if user.data_decline >= 0.3:
        score += 25
    
    # 投诉增加
    if user.complaint_count >= 2:
        score += 25
    
    # 在网时长
    if user.tenure_months <= 12:
        score += 20
    
    return score
```
