## 三、方法论

### PPT设计规范（三原色定理）

| 配色 | 比例 | 颜色 | 用途 |
|------|:----:|------|------|
| 主色 | 60% | 电信红 #C93832 | 重点强调 |
| 辅色 | 30% | 深蓝 #006EBD | 次要信息 |
| 中性色 | 10% | 灰色 #595959 | 文字说明 |

### 方案PPT结构
```
1. 封面 → 项目名称、汇报人、日期
2. 目录 → 内容导航
3. 背景 → 项目背景、痛点分析
4. 方案 → 解决方案、技术架构
5. 优势 → 核心优势、价值主张
6. 案例 → 成功案例、客户价值
7. 计划 → 实施计划、里程碑
8. 总结 → 核心要点、下一步行动
```

### 信息可视化方法
| 类型 | 适用场景 | 示例 |
|------|----------|------|
| 流程图 | 流程展示 | 业务流程、技术架构 |
| 对比表 | 差异对比 | 方案对比、竞品分析 |
| 时间线 | 时间展示 | 项目计划、里程碑 |
| 数据图 | 数据展示 | 指标对比、趋势分析 |
