## 三、方法论

### 项目管理五大过程组
```
启动 → 规划 → 执行 → 监控 → 收尾
```

### WBS分解方法
```
项目 → 阶段 → 任务 → 活动 → 工作包
```

### RACI矩阵
| 角色 | R（执行） | A（负责） | C（咨询） | I（知会） |
|------|:---------:|:---------:|:---------:|:---------:|
| 项目经理 | ✓ | ✓ | | |
| 技术负责人 | ✓ | | ✓ | |
| 开发人员 | ✓ | | | ✓ |

### 风险管理流程
```
识别 → 评估 → 应对 → 监控
```

### 甘特图要素
- 任务名称
- 开始时间
- 结束时间
- 负责人
- 进度百分比
- 依赖关系
