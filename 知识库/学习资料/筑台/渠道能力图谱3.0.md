## 六、赋能模式

### 集团统建模式

```
集团统一建设 → 省份申请使用 → API/账号接入
```

### 集团与省共建模式

```
集团规划标准 → 省份特色建设 → 双方协同运营
```

### 赋能方式

1. **客户端嵌入**：H5、小程序（云溪）
2. **API调用**：接口权限申请
3. **账号配置**：权限管理、数据隔离
4. **能力组合**：原子能力灵活组合

## 七、应用场景

### 1. 客户服务场景

- 查询办理：中国电信APP、数字人
- 咨询服务：企微、AI客服
- 上门服务：电信到家、翼交付

### 2. 营销推广场景

- 内容营销：直播短视频、云溪发布
- 精准营销：企微营销、优惠券平台
- 私域运营：企微、微店

### 3. 商机转化场景

- 商机采集：多触点汇聚
- 商机调度：抢单、派单、转派
- 商机转化：OAO、厅店承接

### 4. 门店运营场景

- 销售管理：门店宝、门店零售
- 数据分析：行为采集、数据看板
- 效能提升：厅店效能管理
