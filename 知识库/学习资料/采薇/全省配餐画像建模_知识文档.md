## 一、用户特征画像维度

### 1.1 五大画像维度

| 画像维度 | 定义 | 应用场景 | 数据来源 |
|----------|------|----------|----------|
| 流量需求度 | 评估用户对移动流量的潜在需求程度 | 流量经营、套餐升级 | 大数据平台 |
| 语音需求度 | 评估用户对通话时长的需求 | 语音包推荐 | 计费系统 |
| 宽带需求度 | 评估用户对宽带速率的需求 | 宽带提速、FTTR | 宽带系统 |
| 价值贡献度 | 评估用户的价值贡献 | VIP客户识别 | 账务系统 |
| 流失风险度 | 评估用户的流失可能性 | 维系挽留 | 大数据模型 |

### 1.2 维度计算规则

**流量需求度**：
```
流量需求度 = (
    套餐流量饱和度 × 0.3 +
    月均流量使用率 × 0.25 +
    流量溢出费用 × 0.25 +
    流量增长趋势 × 0.2
)
```

**语音需求度**：
```
语音需求度 = (
    套餐语音使用率 × 0.3 +
    月均通话时长 × 0.25 +
    语音溢出费用 × 0.25 +
    通话频次 × 0.2
)
```

**宽带需求度**：
```
宽带需求度 = (
    宽带速率等级 × 0.3 +
    光猫类型 × 0.25 +
    是否FTTR × 0.25 +
    宽带使用频次 × 0.2
)
```

## 三、画像应用场景

### 3.1 精准营销

| 场景 | 画像维度 | 推荐产品 |
|------|----------|----------|
| 流量经营 | 流量需求度≥80 | 流量包、升级大流量套餐 |
| 宽带提速 | 宽带需求度≥70 | 千兆提速、FTTR |
| 语音包 | 语音需求度≥60 | 语音包、升级语音套餐 |
| VIP识别 | 价值贡献度≥90 | VIP服务、专属权益 |

### 3.2 流失预警

**流失风险度评估**：
- 高风险：流失风险度≥80 → 立即维系
- 中风险：60≤流失风险度<80 → 定期关怀
- 低风险：流失风险度<60 → 常规维护
