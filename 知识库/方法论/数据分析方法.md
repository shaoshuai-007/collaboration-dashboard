## 二、电信核心指标体系

### 2.1 用户规模指标

| 指标名称 | 计算公式 | 说明 |
|----------|----------|------|
| 在网用户数 | COUNT(DISTINCT user_id) | 当前在网用户总数 |
| 新增用户数 | COUNT(新增用户) | 当月新增用户数 |
| 流失用户数 | COUNT(流失用户) | 当月流失用户数 |
| 净增用户数 | 新增-流失 | 当月净增用户数 |
| 用户增长率 | (新增-流失)/期初用户 | 用户增长比例 |

### 2.2 收入指标

| 指标名称 | 计算公式 | 说明 |
|----------|----------|------|
| ARPU | 总收入/用户数 | 每用户平均收入 |
| AROU | 总收入/订购用户数 | 每订购用户平均收入 |
| 总收入 | SUM(收入金额) | 月度/年度总收入 |
| 收入增长率 | (本期-上期)/上期 | 收入增长比例 |

### 2.3 使用量指标

| 指标名称 | 计算公式 | 说明 |
|----------|----------|------|
| MOU | 总通话时长/用户数 | 每用户月均通话时长 |
| DOU | 总流量/用户数 | 每用户月均流量使用 |
| 短信条数 | SUM(短信条数)/用户数 | 每用户月均短信条数 |

### 2.4 质量指标

| 指标名称 | 计算公式 | 说明 |
|----------|----------|------|
| 接通率 | 接通次数/呼叫次数 | 通话接通成功率 |
| 掉话率 | 掉话次数/通话次数 | 通话掉话比例 |
| 投诉率 | 投诉用户数/总用户数 | 用户投诉比例 |
| 满意度 | 满意用户数/回访用户数 | 用户满意比例 |

## 四、流失预测分析

### 4.1 流失预警模型

```python
# 流失预测特征
churn_features = [
    # 使用量特征
    "avg_mou_3m",          # 近3月平均通话时长
    "avg_dou_3m",          # 近3月平均流量使用
    "mou_decline_rate",    # 通话时长下降率
    "dou_decline_rate",    # 流量使用下降率

    # 消费特征
    "arpu_3m",             # 近3月ARPU
    "balance",             # 当前余额
    "recharge_days",       # 距上次充值天数

    # 互动特征
    "complaint_count",     # 投诉次数
    "service_call_count",  # 客服拨打次数
    "login_days",          # 最近登录天数

    # 套餐特征
    "package_age",         # 套餐使用时长
    "package_expire_days", # 套餐到期天数
]

# 流失预测模型
def predict_churn(user_features):
    """
    预测用户流失概率

    Args:
        user_features: 用户特征字典

    Returns:
        churn_probability: 流失概率 0-1
        risk_level: 风险等级
    """
    # 模型预测逻辑（示例）
    score = 0

    # 使用量下降
    if user_features['mou_decline_rate'] > 0.5:
        score += 30
    if user_features['dou_decline_rate'] > 0.5:
        score += 30

    # 余额不足
    if user_features['balance'] < 10:
        score += 20

    # 长期未充值
    if user_features['recharge_days'] > 30:
        score += 10

    # 投诉次数
    if user_features['complaint_count'] > 3:
        score += 10

    # 计算概率
    churn_probability = min(score / 100, 1.0)

    # 风险等级
    if churn_probability > 0.7:
        risk_level = "高风险"
    elif churn_probability > 0.4:
        risk_level = "中风险"
    else:
        risk_level = "低风险"

    return churn_probability, risk_level
```

### 4.2 流失预警规则

| 预警条件 | 风险等级 | 建议措施 |
|----------|:--------:|----------|

### 5.2 核心指标

```python
# 营销效果指标计算
def calculate_marketing_metrics(campaign_data):
    """
    计算营销效果指标

    Args:
        campaign_data: 营销活动数据

    Returns:
        metrics: 效果指标字典
    """
    metrics = {}

    # 曝光指标
    metrics['impressions'] = campaign_data['exposures']  # 曝光量
    metrics['reach'] = campaign_data['unique_users']     # 曝光人数
    metrics['cpm'] = campaign_data['cost'] / metrics['impressions'] * 1000  # 千次曝光成本

    # 转化指标
    metrics['clicks'] = campaign_data['clicks']          # 点击量
    metrics['ctr'] = metrics['clicks'] / metrics['impressions']  # 点击率
    metrics['conversions'] = campaign_data['conversions']  # 转化量
    metrics['cvr'] = metrics['conversions'] / metrics['clicks']  # 转化率

    # 成本指标
    metrics['cac'] = campaign_data['cost'] / metrics['conversions']  # 获客成本
    metrics['total_revenue'] = campaign_data['revenue']   # 总收入
    metrics['roi'] = (metrics['total_revenue'] - campaign_data['cost']) / campaign_data['cost']  # ROI

    return metrics
```

### 6.2 可视化代码示例

```python
import matplotlib.pyplot as plt
import seaborn as sns
import pandas as pd

# 设置中文显示
plt.rcParams['font.sans-serif'] = ['SimHei']
plt.rcParams['axes.unicode_minus'] = False

# 示例数据
data = pd.DataFrame({
    'month': ['1月', '2月', '3月', '4月', '5月', '6月'],
    'revenue': [100, 120, 115, 130, 125, 140],
    'users': [1000, 1100, 1050, 1200, 1150, 1300]
})

# 趋势图
fig, ax1 = plt.subplots(figsize=(10, 6))

ax1.set_xlabel('月份')
ax1.set_ylabel('收入(万元)', color='tab:red')
ax1.plot(data['month'], data['revenue'], color='tab:red', marker='o', label='收入')
ax1.tick_params(axis='y', labelcolor='tab:red')

ax2 = ax1.twinx()
ax2.set_ylabel('用户数(人)', color='tab:blue')
ax2.plot(data['month'], data['users'], color='tab:blue', marker='s', label='用户数')
ax2.tick_params(axis='y', labelcolor='tab:blue')

plt.title('月度收入与用户数趋势')
fig.tight_layout()
plt.savefig('revenue_trend.png', dpi=300)
```
