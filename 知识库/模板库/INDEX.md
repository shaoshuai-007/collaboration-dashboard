## 六、模板使用指南

### 指南针工程流程

```
步骤1：需求文档 → 采薇负责

步骤5：详细设计 → 工尺负责
步骤6：项目管控 → 玉衡负责
```

### 使用方法

1. **选择模板**：根据任务类型选择对应模板
2. **复制模板**：复制模板内容到工作目录
3. **填写内容**：按照模板结构填写
4. **质量检查**：使用模板中的检查清单验证
