## 🧠 核心创新：2个模型

### 模型1：五维评估模型

```
┌─────────────────────────────────────────────────────────────────┐
│                    五维评估模型                                  │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  ┌───────────────────────────────────────────────────────────┐ │
│  │  维度      权重    推导依据                                │ │
│  ├───────────────────────────────────────────────────────────┤ │
│  │  准确性    25%    商务谈判场景占比40%                      │ │
│  │  价值度    25%    商务谈判+客户拜访场景占比60%             │ │
│  │  可用性    30%    方案撰写场景占比30%（最高权重）          │ │
│  │  时效衰减  10%    客户拜访场景占比20%                      │ │
│  │  稀缺性    10%    竞品分析场景占比10%                      │ │
│  └───────────────────────────────────────────────────────────┘ │
│                                                                 │
│  核心创新：                                                     │
│  ─────────────────────────────────────────────────────────────│
│  · 基于场景推导权重，避免主观臆断                               │
│  · 可用性权重最高（30%），方案撰写是核心场景                    │
│  · 时效衰减按情报类型区分（政策/技术/案例/竞品）                │
│                                                                 │
│  分级标准：                                                     │
│  ─────────────────────────────────────────────────────────────│
│  S级（≥80分）→ 精华库 + QQ推送                                 │
│  A级（70-79分）→ 精华库                                        │
│  B级（60-69分）→ 原始库                                        │
│  C级（50-59分）→ 原始库（低优先）                              │
│  D级（<50分）→ 舍弃                                            │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```
