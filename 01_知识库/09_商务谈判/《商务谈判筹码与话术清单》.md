# 商务谈判筹码与话术清单

## 价格让步交换资源
- 降价 X% → 换取客户免费提供数据接口使用权
- 赠送半年维保 → 换取客户同意作为成功案例宣传
- 延长付款周期 → 换取客户提前签署验收单

## 针对不同采购风格的话术
- **技术型**：强调算法精度、模型可解释性、技术架构先进性。
- **关系型**：强调过往合作成功案例、团队服务意识、长期共赢。
- **价格敏感型**：强调总拥有成本低、投资回报周期短、可分期付款。

## 常见反对意见应对
- “你们太贵了” → “我们的方案能帮您降低离网率X%，带来的收益远超成本。”
- “我们再考虑一下” → “理解，请问您主要担心哪方面？我们可以提供更详细的数据支持。”
