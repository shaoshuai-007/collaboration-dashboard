## 🔗 方法论组合使用

### 问题到方案全流程

```
5W2H + SWOT → SMART + OKR → 六顶思考帽 → KANO + 干系人地图 → WBS + RACI → 成本收益 + 风险矩阵 → 金字塔原理 + SCQA
```

### 快速诊断问题

```
5W2H（定义）→ 根因分析（深挖）→ 鱼骨图（可视化）
```

### 需求分析组合

```
干系人地图（找对人）→ 用户故事（说对话）→ KANO（排好序）→ MoSCoW（定优先）
```

### 方案撰写组合

```
金字塔原理（搭骨架）→ SCQA（写开场）→ MECE（查遗漏）
```
