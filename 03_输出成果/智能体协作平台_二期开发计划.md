### 任务1：方案导出（3天）

**目标**：将Agent讨论结果导出为Word/Excel文档

**开发内容**：
1. 讨论结果结构化存储
2. Word文档生成（python-docx）
3. Excel表格生成（openpyxl）
4. 导出接口开发

**产出物**：
- 导出API接口
- Word模板库
- Excel模板库

**技术方案**：
```python
# 后端：Flask API
@app.route('/api/export/word', methods=['POST'])
def export_word():
    discussion = request.json.get('discussion')
    doc = generate_word_document(discussion)
    return send_file(doc, as_attachment=True)

# 模板引擎：jinja2 + python-docx
```
