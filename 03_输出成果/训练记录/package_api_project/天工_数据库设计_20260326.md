## 一、核心数据表设计

### 1.1 用户表（tb_user）

| 字段名 | 类型 | 说明 | 约束 |
|--------|------|------|------|
| user_id | BIGINT | 用户ID | PK |
| phone | VARCHAR(11) | 手机号 | UNIQUE, NOT NULL |
| name | VARCHAR(50) | 姓名 | |
| id_card | VARCHAR(18) | 身份证号 | |
| address | VARCHAR(200) | 地址 | |
| user_level | TINYINT | 用户等级（1-5） | DEFAULT 1 |
| create_time | DATETIME | 创建时间 | DEFAULT CURRENT_TIMESTAMP |
| update_time | DATETIME | 更新时间 | DEFAULT CURRENT_TIMESTAMP ON UPDATE |

### 1.2 套餐表（tb_package）

| 字段名 | 类型 | 说明 | 约束 |
|--------|------|------|------|
| package_id | INT | 套餐ID | PK |
| package_name | VARCHAR(100) | 套餐名称 | NOT NULL |
| package_type | VARCHAR(20) | 套餐类型（5G/宽带/融合） | NOT NULL |
| monthly_fee | DECIMAL(10,2) | 月费 | NOT NULL |
| data_quota | INT | 流量配额（MB） | |
| voice_quota | INT | 语音配额（分钟） | |
| bandwidth | INT | 宽带带宽（Mbps） | |
| status | TINYINT | 状态（0下架/1上架） | DEFAULT 1 |
| create_time | DATETIME | 创建时间 | DEFAULT CURRENT_TIMESTAMP |

### 1.3 订单表（tb_order）

| 字段名 | 类型 | 说明 | 约束 |
|--------|------|------|------|
| order_id | BIGINT | 订单ID | PK |
| user_id | BIGINT | 用户ID | FK |
| package_id | INT | 套餐ID | FK |
| order_type | TINYINT | 订单类型（1新装/2续约/3变更） | NOT NULL |
| order_status | TINYINT | 订单状态 | DEFAULT 1 |
| order_amount | DECIMAL(10,2) | 订单金额 | NOT NULL |
| pay_time | DATETIME | 支付时间 | |
| effect_time | DATETIME | 生效时间 | |
| expire_time | DATETIME | 过期时间 | |
| create_time | DATETIME | 创建时间 | DEFAULT CURRENT_TIMESTAMP |

### 1.4 用户套餐表（tb_user_package）

| 字段名 | 类型 | 说明 | 约束 |
|--------|------|------|------|
| id | BIGINT | ID | PK |
| user_id | BIGINT | 用户ID | FK |
| package_id | INT | 套餐ID | FK |
| start_time | DATETIME | 开始时间 | NOT NULL |
| end_time | DATETIME | 结束时间 | |
| status | TINYINT | 状态（1生效/2过期/3退订） | DEFAULT 1 |

## 二、分表策略

### 2.1 按月分表（tb_order_YYYYMM）

订单表按月份分表：
- tb_order_202601
- tb_order_202602
- ...

### 2.2 分表规则

```sql
-- 获取分表名
function get_order_table($date) {
    return 'tb_order_' . date('Ym', strtotime($date));
}
```
