## 二、技能学习（15分钟）

### 2.1 ai-ppt-generator技能介绍

**技能名称**：ai-ppt-generator
**技能特点**：百度AI智能PPT生成，模板智能选择

**8种模板风格**：

| 风格 | 适用场景 |
|------|----------|
| 企业商务 | 商业汇报、企业方案 |
| 未来科技 | 技术方案、科技产品 |
| 卡通手绘 | 教育培训、儿童内容 |
| 创意趣味 | 创意提案、活动策划 |
| 中国风 | 传统文化、政府汇报 |
| 年终总结 | 年度汇报、总结分析 |
| 扁平简约 | 简洁汇报、数据展示 |
| 文艺清新 | 文化艺术、品牌传播 |

### 2.2 技能调用流程

```
用户输入主题
    ↓
询问是否选择模板
    ├── 是 → 展示模板列表 → 用户选择 → 生成PPT
    └── 否 → 智能推荐模板 → 自动生成PPT
    ↓
等待生成（2-5分钟）
    ↓
获取PPT下载链接
```

### 2.3 技能命令

```bash
# 查看所有模板
python3 scripts/ppt_theme_list.py

# 智能自动选择（推荐）
python3 scripts/random_ppt_theme.py --query "AI智能配案系统方案"

# 指定模板
python3 scripts/generate_ppt.py --query "AI智能配案系统方案" --tpl_id 106
```

## 三、内容设计（45分钟）

### 3.1 PPT大纲设计

**主题**：AI智能配案系统方案
**目标受众**：湖北电信管理层
**设计风格**：企业商务 + 未来科技

**内容大纲**：

```
一、项目背景（2页）
├── 业务痛点
│   - 套餐复杂度高，人工配案效率低
│   - 129个场景，41个标签，配置难度大
│   - 有效率仅4.1%，资源浪费严重
└── 建设目标
    - 提升配案效率
    - 提高套餐匹配精准度
    - 降低人工成本

二、系统方案（4页）
├── 系统架构
│   - AI引擎层
│   - 数据处理层
│   - 应用服务层
├── 核心能力
│   - 智能标签识别
│   - 套餐匹配推荐
│   - 场景化适配
└── 技术路线
    - 深度学习模型
    - 规则引擎
    - 实时计算

三、功能演示（3页）
├── 用户画像洞察
├── 套餐智能推荐
└── 配案效果评估

四、实施计划（2页）
├── 分阶段实施
│   - 第1阶段：数据准备
│   - 第2阶段：模型训练
│   - 第3阶段：试点上线
│   - 第4阶段：全省推广
└── 资源需求

五、价值评估（2页）
├── 效率提升：人工30分钟 → AI 3秒
├── 准确率提升：60% → 85%
└── 成本降低：人力成本降低50%

六、成功案例（1页）
└── XX省电信成功实践
```
