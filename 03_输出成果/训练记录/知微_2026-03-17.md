## 一、电信核心指标体系总结

### 1. ARPU（每用户平均收入）
- **公式**: ARPU = 总收入 / 用户数
- **意义**: 衡量用户价值的核心指标，反映用户的付费能力和价值贡献
- **应用**: 用于评估用户价值等级、制定差异化服务策略

### 2. MOU（每用户平均通话时长）
- **公式**: MOU = 总通话时长 / 用户数
- **单位**: 分钟/月
- **意义**: 反映用户的语音使用习惯和活跃度
- **应用**: 套餐推荐依据、用户行为分析

### 3. DOU（每用户平均流量使用）
- **公式**: DOU = 总流量 / 用户数
- **单位**: MB或GB/月
- **意义**: 反映用户的数据消费习惯，是5G时代关键指标
- **应用**: 流量包推荐、套餐适配分析

### 4. 用户留存率与流失率
- **留存率**: 当期末仍在网用户 / 期初用户 × 100%
- **流失率**: 当期流失用户 / 期初用户 × 100%
- **意义**: 衡量用户忠诚度和运营健康度
- **预警阈值**: 
  - 话费余额 < 10元 → 中风险
  - 通话/流量使用下降 > 50% → 高风险
  - 投诉次数 > 3次/月 → 高风险

### 5. 套餐渗透率与转化率
- **渗透率**: 某套餐订购用户数 / 总用户数 × 100%
- **转化率**: 营销活动转化用户数 / 触达用户数 × 100%
- **意义**: 衡量产品推广效果和用户接受度
- **应用**: 营销效果评估、产品优化方向

### 6. 用户增长率（补充指标）
- **公式**: (新增用户 - 流失用户) / 期初用户 × 100%
- **意义**: 反映业务发展态势

### 7. 接通率与投诉率（质量指标）
- **接通率**: 接通次数 / 呼叫次数 × 100%
- **投诉率**: 投诉用户数 / 总用户数 × 100%
- **意义**: 衡量服务质量和用户满意度

## 二、数据分析方法要点提炼

### 方法一：数据分析流程（六步法）

```
1. 明确问题 → 业务理解、目标定义、指标确定
2. 数据获取 → 数据源识别、数据采集、数据存储
3. 数据清洗 → 缺失值处理、异常值处理、数据标准化
4. 数据分析 → 描述性/诊断性/预测性分析
5. 数据可视化 → 图表设计、仪表盘制作、报告生成
6. 结论建议 → 发现总结、建议方案、行动计划
```

**关键要点**:
- 分析方法分四类：描述性（发生了什么）、诊断性（为什么发生）、预测性（将发生什么）、规范性（应该怎么做）
- 每个阶段都需要业务人员参与验证

### 方法二：用户画像构建方法

**画像维度结构**:
```
用户画像
├── 人口属性: 年龄、性别、地域、职业
├── 行为特征: 通话行为、上网行为、消费行为
├── 心理特征: 价格敏感度、品牌偏好、风险偏好
└── 社交特征: 社交圈大小、社交活跃度、影响力指数
```

**标签体系示例**:
- 基础标签: age_range, gender, city_level
- 行为标签: call_preference, data_usage, app_preference
- 价值标签: arpu_level, value_score, churn_risk
- 营销标签: price_sensitivity, upgrade_potential, cross_sell

**关键要点**:
- 画像标签需要定期更新
- 标签粒度要平衡可解释性与精准度

### 方法三：流失预测方法论

**核心特征**:
| 特征类别 | 关键指标 |
|----------|----------|
| 使用量特征 | 近3月平均通话时长、流量使用、下降率 |
| 消费特征 | ARPU、当前余额、距上次充值天数 |
| 互动特征 | 投诉次数、客服拨打次数、最近登录天数 |
| 套餐特征 | 套餐使用时长、套餐到期天数 |

**预警规则**:
- 连续2个月ARPU下降>30% → 高风险
- 余额<10元且7天未充值 → 高风险
- 通话时长下降>50% → 高风险

**关键要点**:
- 需要建立多维度预警体系
- 预警后要有明确的干预措施

## 三、学习心得

### 今日收获
1. **系统性认知**: 电信业务不只是"打电话上网"，而是包含用户管理、计费结算、套餐管理、存量经营等完整体系
2. **指标关联性**: ARPU、MOU、DOU等指标不是孤立的，它们共同构成用户价值画像
3. **数据驱动决策**: 用户画像和流失预测都是将数据转化为行动指南的典型应用

### 关键发现
- AI智能配案流程涉及用户画像构建→需求分析→套餐匹配→推荐展示→效果跟踪的完整闭环
- 用户生命周期管理分为新用户期→成长期→成熟期→衰退期→流失期五个阶段
- 流失预警需要结合使用量、消费、互动、套餐四类特征综合判断
