## 视觉风格规范

### 配色方案
| 用途 | 色值 | RGB | 应用场景 |
|------|------|-----|----------|
| 主色：科技蓝 | #006EBD | 0, 110, 189 | 背景、主标题、数据图形、品牌元素 |
| 辅色：NVIDIA绿 | #76B900 | 118, 185, 0 | 高亮、按钮、进度条、积极数据 |
| 点缀色：数据红 | #C93832 | 201, 56, 50 | 警示数据、关键词强调、重点标注 |

### 字体规范
- **主标题**: 思源黑体 Heavy / 72-96px / 科技蓝
- **数据强调**: DIN Pro Bold / 120-200px / 渐变色（蓝→绿）
- **正文**: 思源黑体 Medium / 36-48px / 白色
- **字幕**: 思源黑体 Regular / 32-40px / 白色+黑色描边

### 视觉元素
- **品牌logo**: NVIDIA绿logo + "GTC 2026"白字
- **数据动画**: 柱状图、数字滚动、粒子流动
- **转场**: 网格扭曲、粒子消散、滑动切换
- **背景**: 深蓝渐变 + 网格线 + 光晕效果

## 关键数据可视化规范

### 数据类型与表现形式

| 数据类型 | 表现形式 | 配色 | 动画方式 |
|----------|----------|------|----------|
| 增长倍数 | 数字爆炸 | 数据红 + 白 | 滚动计数 → 爆炸脉冲 |
| 金额数据 | 柱状图 | 科技蓝→NVIDIA绿渐变 | 高度增长 + 粒子流 |
| 性能指标 | 浮动标签 | 科技蓝/绿/红分类 | 依次淡入 + 光晕 |
| 时间对比 | 双柱/进度条 | 蓝（过去）vs 绿（未来） | 左右展开 |
| 产品参数 | 参数卡片 | 白字 + 图标 | 滑入 + 高亮 |
| 公式演算 | 数学公式 | 白色 + 等号绿 | 逐步计算动画 |
| 时间轴 | 横向时间线 | 渐变 + 虚线未来 | 从左到右展开 |
| 层级结构 | 同心圆/金字塔 | 蓝/绿/红分层 | 点击高亮 |

### 动画节奏规范

- **快节奏镜头**（2-3秒）：开篇钩子、片头片尾
- **中节奏镜头**（5-8秒）：核心数据、产品展示
- **慢节奏镜头**（8-10秒）：概念解释、未来展望
- **转场时长**：0.5-1秒

## 设计理念总结

本分镜设计以"震撼数据 + 视觉冲击"为核心策略，通过以下方式实现传播目标：

1. **开篇炸场**：100万倍数字爆炸，瞬间抓住注意力
2. **数据可视化**：柱状图、时间轴、公式动画，让抽象数据具象化
3. **品牌色彩**：科技蓝+ NVIDIA绿+ 数据红，强化品牌识别
4. **节奏把控**：快慢结合，关键数据放慢强调
5. **结尾升华**：Token经济概念，留下思考和讨论空间

**传播亮点设计**：
- "100万倍"成为记忆锚点
- "两行命令"激发技术好奇心
- "ChatGPT时刻"类比降低认知门槛
- "Token工厂"概念引发讨论
