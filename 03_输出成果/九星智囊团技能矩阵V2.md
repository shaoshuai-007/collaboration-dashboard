# 九星智囊团技能矩阵 V2.0

## 设计理念

**一个成员 = 多个技能 = 角色闭环**

每个成员根据自己的角色定位，调用多个技能完成端到端任务。

---

## 成员技能矩阵

### 🌸 采薇 - 需求分析师

| 技能 | 用途 | 触发场景 |
|------|------|---------|
| summarize | 内容摘要、关键点提取 | 提取需求要点 |
| compass-needdoc | 需求文档生成 | 生成需求文档 |
| compass-mindmap | 思维导图生成 | 分析需求结构 |
| spreadsheet | Excel处理 | 整理需求清单 |
| document-pdf | PDF处理 | 读取客户提供的PDF |
| send-email | 邮件发送 | 发送需求确认邮件 |

**典型工作流**：
```
客户需求 → summarize提取要点 → compass-needdoc生成文档
         → compass-mindmap分析结构 → spreadsheet整理清单
         → send-email发送确认
```

## 技能调用优先级

每个成员调用技能的优先级：

1. **compass-* 系列技能**（指南针工程专用）
2. **角色核心技能**（如coding-agent、summarize）
3. **辅助技能**（如spreadsheet、send-email）
4. **展示技能**（如canvas、infographic-creator）
