**项目名称**：集团大数据平台Token采集系统
**方案版本**：V1.0
**编制时间**：2026年3月18日
**编制团队**：九星智囊团

## 一、方案概述

### 1.1 项目背景

中国电信集团各单位已部署大量AI应用和大模型，Token消耗数据分散在各系统，缺乏统一采集和管控。本项目旨在构建集团级Token采集系统，实现全集团AI应用Token消耗数据的统一采集、统一存储、统一分析、统一展示。

### 1.2 建设目标

| 目标层级 | 目标内容 | 衡量指标 |
|----------|----------|----------|
| 数据采集 | 56个单位Token数据统一采集 | 采集覆盖率100% |
| 数据存储 | 建立集团级Token数据仓库 | 数据完整性100% |
| 数据分析 | 多维度统计分析能力 | 分析维度≥10个 |
| 数据展示 | 可视化大屏和报表 | 展示指标≥50个 |
| 运维管理 | 自动化运维和监控 | 故障恢复<30分钟 |

### 1.3 建设范围

| 范围维度 | 具体内容 | 数量 |
|----------|----------|:----:|
| 采集单位 | 总部部门、专业公司、省公司、直属机构 | 56+ |
| AI应用 | 智能服务、智能营销、智能运营、智能管理等 | 76类 |
| 大模型 | DeepSeek、Qwen、Telechat、Llama、Baichuan等 | 45+ |
| 部署硬件 | 910B、H800、H100、A100、V100等 | 13种 |
| Token类型 | 输入Token、输出Token、Thinking Token | 3类 |

#### 3.4.2 编码规则详细说明

**模型编码规则**：
```
格式：{模型部署单位标识}_{模型品牌}_{模型参数量}_{序号}

组成部分：
- 模型部署单位标识：见主数据（如YWYYB、811等）
- 模型品牌：大写字母（如QWEN、DEEPSEEK、TELECHAT）
- 模型参数量：数值（如72、7、1.5）
- 序号：六位数字（如000001）

约束：
- 全部使用大写字母及数字
- 各部分用英文下划线"_"连接
- 一条数据仅允许填写一个模型的编码

示例：
- YWYYB_QWEN_72_000001
- 811_TELECHAT_12_000001
- SHCHB_DEEPSEEK_7_000001
```

**应用编码规则**：
```
格式：{AI应用业务归属单位标识}_{AI应用图谱编码}_{序号}

组成部分：
- AI应用业务归属单位标识：见主数据
- AI应用图谱编码：见主数据（76个编码）
- 序号：六位数字

约束：
- AI应用图谱编码需对应到三级分类
- 无法归类到三级分类的，对应到一级分类的"其他"

示例：
- YWYYB_ZNYY0001_000001
- SHCHB_ZNFW0001_000001
- 811_BGWJ0000_000001
```

### 3.5 回执文件规范

#### 3.5.1 CHECK回执

**成功回执**（后缀.RPT）：
```
{文件名称}[0x05]00[0x0D0A]
```

**失败回执**（后缀.ERR）：
```
{文件名称}[0x05]{错误码}[0x0D0A]
```

**CHECK错误码**：

| 错误码 | 说明 | 处理建议 |
|:------:|------|----------|
| 00 | 核查成功 | - |
| 10001 | CHECK文件无法打开 | 检查文件格式和权限 |
| 10002 | 接口数据文件不存在 | 检查文件是否上传 |
| 10003 | 数据文件名异常 | 检查命名规范 |
| 10004 | CHECK列表中无此文件 | 检查CHECK文件内容 |
| 10005 | CHECK文件上传超时 | 检查网络连接 |
| 10099 | 规定时间内无CHECK文件 | 检查上传流程 |

#### 3.5.2 VAL回执

**成功回执**（后缀.RPT）：
```
{文件名称}[0x05]{处理时间}[0x05]00[0x0D0A]
```

**失败回执**（后缀.ERR）：
```
{文件名称}[0x05]{处理时间}[0x05]{错误码}[0x0D0A]
```

**VAL错误码**：

| 错误码 | 说明 | 处理建议 |
|:------:|------|----------|
| 00 | 校验成功 | - |
| 20001 | 数据文件无法打开 | 检查文件格式和权限 |
| 20002 | 字段个数异常 | 检查数据格式 |
| 20003 | 字段顺序异常 | 检查字段顺序 |
| 20004 | 数据为空 | 检查数据生成逻辑 |
| 20005 | 文件大小不符 | 检查压缩方式 |
| 20006 | 文件记录数不符 | 检查记录数统计 |
| 20007 | MD5校验码不符 | 重新生成VAL文件 |
| 20008 | VAL文件不存在 | 检查上传流程 |
| 20009 | 数据文件不存在 | 检查文件是否上传 |

**文件校验器**：

```python
# file_validator.py
"""
文件校验器
"""

import gzip
import hashlib
from pathlib import Path
from typing import Tuple, List
from loguru import logger


class FileValidator:
    """文件校验器"""
    
    def validate_check(self, check_file: Path) -> Tuple[bool, str, List[str]]:
        """
        校验CHECK文件
        
        Returns:
            (是否成功, 错误码, 文件列表)
        """
        try:
            # 读取CHECK文件
            files = []
            with open(check_file, 'r', encoding='utf-8') as f:
                for line in f:
                    parts = line.strip().split('\x05')
                    if parts:
                        files.append(parts[0])
                        
            logger.info(f"CHECK文件包含{len(files)}个文件")
            
            # 检查文件是否存在
            missing_files = []
            for file in files:
                file_path = check_file.parent / file
                if not file_path.exists():
                    missing_files.append(file)
                    
            if missing_files:
                logger.error(f"缺失文件: {missing_files}")
                return False, '10002', files
                
            return True, '00', files
            
        except Exception as e:
            logger.error(f"CHECK文件校验异常: {e}")
            return False, '10001', []
            
    def validate_val(
        self, 
        val_file: Path, 
        data_file: Path
    ) -> Tuple[bool, str]:
        """
        校验VAL文件
        
        Returns:
            (是否成功, 错误码)
        """
        try:
            # 读取VAL文件
            with open(val_file, 'r', encoding='utf-8') as f:
                parts = f.readline().strip().split('\x05')
                
            if len(parts) < 7:
                logger.error(f"VAL文件字段个数异常: {len(parts)}")
                return False, '20002'
                
            filename = parts[0]
            record_count = int(parts[1])
            file_size = int(parts[2])
            md5 = parts[3]
            
            logger.info(f"VAL校验: {filename}, 记录数={record_count}, 大小={file_size}")
            
            # 校验文件大小
            actual_size = data_file.stat().st_size
            if actual_size != file_size:
                logger.error(f"文件大小不符: 预期{file_size}, 实际{actual_size}")
                return False, '20005'
                
            # 校验MD5
            actual_md5 = self._calculate_md5(data_file)
            if actual_md5 != md5:
                logger.error(f"MD5校验失败: 预期{md5}, 实际{actual_md5}")
                return False, '20007'
                
            # 校验记录数
            actual_count = self._count_records(data_file)
            if actual_count != record_count:
                logger.error(f"记录数不符: 预期{record_count}, 实际{actual_count}")
                return False, '20006'
                
            logger.info("VAL校验通过")
            return True, '00'
            
        except Exception as e:
            logger.error(f"VAL文件校验异常: {e}")
            return False, '20001'
            
    def validate_data(self, data_file: Path) -> Tuple[bool, str, int]:
        """
        校验数据文件
        
        Returns:
            (是否成功, 错误码, 有效记录数)
        """
        try:
            valid_count = 0
            invalid_records = []
            
            with gzip.open(data_file, 'rt', encoding='utf-8') as f:
                for line_num, line in enumerate(f, 1):
                    line = line.strip()
                    if not line:
                        continue
                        
                    parts = line.split('\x05')
                    
                    # 校验字段个数
                    if len(parts) != 21:
                        invalid_records.append((line_num, '字段个数异常'))
                        continue
                        
                    # 校验主键非空
                    if not all([parts[0], parts[2], parts[4], parts[6]]):
                        invalid_records.append((line_num, '主键为空'))
                        continue
                        
                    valid_count += 1
                    
            if valid_count == 0:
                logger.error("数据为空")
                return False, '20004', 0
                
            if invalid_records:
                logger.warning(f"无效记录: {len(invalid_records)}条")
                
            logger.info(f"数据校验通过: 有效记录{valid_count}条")
            return True, '00', valid_count
            
        except Exception as e:
            logger.error(f"数据文件校验异常: {e}")
            return False, '20001', 0
            
    def _calculate_md5(self, file_path: Path) -> str:
        """计算文件MD5"""
        hash_md5 = hashlib.md5()
        with open(file_path, 'rb') as f:
            for chunk in iter(lambda: f.read(8192), b''):
                hash_md5.update(chunk)
        return hash_md5.hexdigest()
        
    def _count_records(self, gz_file: Path) -> int:
        """统计记录数"""
        count = 0
        with gzip.open(gz_file, 'rt', encoding='utf-8') as f:
            for line in f:
                if line.strip():
                    count += 1
        return count
```

# token_collector.py
"""
Token日志采集器
"""

import json
import gzip
import hashlib
from datetime import datetime, date
from pathlib import Path
from typing import Dict, List, Optional
from dataclasses import dataclass
import pandas as pd
from loguru import logger


@dataclass
class TokenRecord:
    """Token记录"""
    app_owner_org: str
    app_owner_department: str
    model_deploy_org: str
    model_deploy_department: str
    model_code: str
    model_name: str
    app_id: str
    use_case: str
    iternal_app: str
    model_brand: str
    model_size: float
    model_type: str
    quant_level: str
    hardware_type: str
    total_call_count: int
    total_call_success_count: int
    input_token_count: int
    output_token_count: int
    reasoning_token_count: int
    total_token_count: int
    remark: str


class TokenCollector:
    """Token日志采集器"""
    
    def __init__(
        self,
        org_code: str,
        system_code: str,
        master_data: Dict
    ):
        """
        初始化
        
        Args:
            org_code: 单位编码
            system_code: 系统编码（4位英文）
            master_data: 主数据字典
        """
        self.org_code = org_code
        self.system_code = system_code
        self.master_data = master_data
        self.records: List[TokenRecord] = []
        
    def parse_log(self, log_file: str, log_format: str = 'json'):
        """
        解析调用日志
        
        Args:
            log_file: 日志文件路径
            log_format: 日志格式（json/csv/custom）
        """
        logger.info(f"解析日志文件: {log_file}")
        
        if log_format == 'json':
            self._parse_json_log(log_file)
        elif log_format == 'csv':
            self._parse_csv_log(log_file)
        else:
            self._parse_custom_log(log_file)
            
        logger.info(f"解析完成，共{len(self.records)}条记录")
        
    def _parse_json_log(self, log_file: str):
        """解析JSON格式日志"""
        with open(log_file, 'r', encoding='utf-8') as f:
            for line_num, line in enumerate(f, 1):
                try:
                    log = json.loads(line.strip())
                    record = self._extract_token_info(log)
                    if record:
                        self.records.append(record)
                except Exception as e:
                    logger.warning(f"解析失败（行{line_num}）: {e}")
                    continue
                    
    def _extract_token_info(self, log: Dict) -> Optional[TokenRecord]:
        """从日志中提取Token信息"""
        try:
            # 根据实际日志格式调整
            return TokenRecord(
                app_owner_org=log.get('app_owner_org', self.org_code),
                app_owner_department=log.get('app_owner_department', ''),
                model_deploy_org=log.get('model_deploy_org', self.org_code),
                model_deploy_department=log.get('model_deploy_department', ''),
                model_code=log.get('model_code', ''),
                model_name=log.get('model_name', ''),
                app_id=log.get('app_id', ''),
                use_case=log.get('use_case', ''),
                iternal_app=log.get('iternal_app', '1'),
                model_brand=log.get('model_brand', '').upper(),
                model_size=float(log.get('model_size', 0)),
                model_type=log.get('model_type', '1'),
                quant_level=log.get('quant_level', '2'),
                hardware_type=log.get('hardware_type', ''),
                total_call_count=int(log.get('total_call_count', 0)),
                total_call_success_count=int(log.get('total_call_success_count', 0)),
                input_token_count=int(log.get('input_token_count', 0)),
                output_token_count=int(log.get('output_token_count', 0)),
                reasoning_token_count=int(log.get('reasoning_token_count', 0)),
                total_token_count=int(log.get('total_token_count', 0)),
                remark=log.get('remark', '')
            )
        except Exception as e:
            logger.warning(f"提取Token信息失败: {e}")
            return None
            
    def aggregate_by_model_app(self) -> pd.DataFrame:
        """按模型和应用聚合"""
        if not self.records:
            logger.warning("无记录可聚合")
            return pd.DataFrame()
            
        # 转换为DataFrame
        df = pd.DataFrame([vars(r) for r in self.records])
        
        # 聚合
        agg_df = df.groupby([
            'app_owner_org', 'app_owner_department',
            'model_deploy_org', 'model_deploy_department',
            'model_code', 'model_name', 'app_id', 'use_case',
            'iternal_app', 'model_brand', 'model_size',
            'model_type', 'quant_level', 'hardware_type'
        ]).agg({
            'total_call_count': 'sum',
            'total_call_success_count': 'sum',
            'input_token_count': 'sum',
            'output_token_count': 'sum',
            'reasoning_token_count': 'sum',
            'total_token_count': 'sum',
            'remark': 'first'
        }).reset_index()
        
        logger.info(f"聚合完成，共{len(agg_df)}条记录")
        return agg_df
        
    def generate_data_file(
        self, 
        data_date: date, 
        output_dir: str,
        batch: str = '00'
    ) -> str:
        """
        生成数据文件
        
        Returns:
            数据文件路径
        """
        output_path = Path(output_dir)
        output_path.mkdir(parents=True, exist_ok=True)
        
        # 文件名
        file_time = datetime.now().strftime('%Y%m%d')
        filename = f"10800_{self.org_code}_{self.system_code}_AIP_MDL_TOKEN_INFO_{file_time}_{data_date.strftime('%Y%m%d')}_D_{batch}_0001"
        
        # 聚合数据
        df = self.aggregate_by_model_app()
        
        if df.empty:
            logger.warning("无数据，生成空文件")
            
        # 写入数据文件
        data_file = output_path / f"{filename}.DAT"
        with open(data_file, 'w', encoding='utf-8') as f:
            for _, row in df.iterrows():
                line = '\x05'.join([
                    str(row['app_owner_org']),
                    str(row['app_owner_department']),
                    str(row['model_deploy_org']),
                    str(row['model_deploy_department']),
                    str(row['model_code']),
                    str(row['model_name']),
                    str(row['app_id']),
                    str(row['use_case']),
                    str(row['iternal_app']),
                    str(row['model_brand']),
                    str(row['model_size']),
                    str(row['model_type']),
                    str(row['quant_level']),
                    str(row['hardware_type']),
                    str(row['total_call_count']),
                    str(row['total_call_success_count']),
                    str(row['input_token_count']),
                    str(row['output_token_count']),
                    str(row['reasoning_token_count']),
                    str(row['total_token_count']),
                    str(row['remark'])
                ])
                f.write(line + '\r\n')
                
        # 压缩
        gz_file = f"{data_file}.gz"
        with open(data_file, 'rb') as f_in:
            with gzip.open(gz_file, 'wb') as f_out:
                f_out.writelines(f_in)
                
        # 删除原文件
        data_file.unlink()
        
        logger.info(f"数据文件生成完成: {gz_file}")
        return gz_file
        
    def generate_val_file(
        self, 
        data_file: str, 
        data_date: date,
        output_dir: str
    ) -> str:
        """生成VAL校验文件"""
        gz_path = Path(data_file)
        output_path = Path(output_dir)
        
        # 计算MD5
        md5 = self._calculate_md5(gz_path)
        
        # 获取文件大小
        file_size = gz_path.stat().st_size
        
        # 统计记录数
        record_count = self._count_records(gz_path)
        
        # VAL文件名
        val_filename = gz_path.name.replace('.DAT.gz', '.VAL')
        val_file = output_path / val_filename
        
        # 写入VAL文件
        with open(val_file, 'w', encoding='utf-8') as f:
            fields = [
                gz_path.name,                           # 数据文件名称
                str(record_count),                      # 文件记录数
                str(file_size),                         # 文件大小
                md5,                                    # MD5字符
                data_date.strftime('%Y%m%d'),          # 数据日期
                'D',                                    # 上传周期
                datetime.now().strftime('%Y%m%d%H%M%S') # 生成时间
            ]
            f.write('\x05'.join(fields) + '\r\n')
            
        logger.info(f"VAL文件生成完成: {val_file}")
        return str(val_file)
        
    def generate_check_file(
        self, 
        files: List[str], 
        output_dir: str
    ) -> str:
        """生成CHECK清单文件"""
        output_path = Path(output_dir)
        
        # CHECK文件名（取第一个文件的命名规则）
        first_file = Path(files[0])
        check_filename = first_file.name.replace('_0001.DAT.gz', '_0001.CHECK')
        check_file = output_path / check_filename
        
        # 写入CHECK文件
        with open(check_file, 'w', encoding='utf-8') as f:
            for file in files:
                f.write(f"{Path(file).name}\x05D\r\n")
                
        logger.info(f"CHECK文件生成完成: {check_file}")
        return str(check_file)
        
    def _calculate_md5(self, file_path: Path) -> str:
        """计算文件MD5"""
        hash_md5 = hashlib.md5()
        with open(file_path, 'rb') as f:
            for chunk in iter(lambda: f.read(8192), b''):
                hash_md5.update(chunk)
        return hash_md5.hexdigest()
        
    def _count_records(self, gz_file: Path) -> int:
        """统计记录数"""
        count = 0
        with gzip.open(gz_file, 'rt', encoding='utf-8') as f:
            for line in f:
                if line.strip():
                    count += 1
        return count
