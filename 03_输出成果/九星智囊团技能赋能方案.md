## 三、分角色赋能方案

### 🌸 采薇（需求分析师）

**核心赋能**：🧾 summarize（⭐⭐⭐）

**应用场景**：

1. **需求文档快速理解**
```bash
summarize "客户需求文档.pdf" --length xl --model google/gemini-3-flash-preview
```

2. **会议纪要自动生成**
```bash
summarize "会议录音链接" --length medium
```

3. **竞品分析报告**
```bash
summarize "竞品官网URL" --length xl
```

**赋能效果**：
- 需求理解速度提升 5倍
- 文档分析准确率 90%+
- 会议纪要生成自动化
