### 状态色系

| 状态 | 颜色 | 用途 |
|------|------|------|
| 🟢 运行中 | `#00D68F` | Agent正在执行任务 |
| 🟢 完成 | `#00D68F` | 任务执行成功 |
| 🟡 空闲 | `#FFB800` | Agent待命中 |
| 🔴 错误 | `#FF3D71` | 任务执行失败 |
| 🔵 信息 | `#006EBD` | 普通日志信息 |
| ⚪ 禁用 | `#5A6B82` | Agent离线/禁用 |

### 渐变效果

```css
/* 进度条渐变 */
background: linear-gradient(90deg, #C93832 0%, #FF6B6B 100%);

/* 卡片边框光效 */
border: 1px solid rgba(0, 110, 189, 0.3);
box-shadow: 0 0 20px rgba(201, 56, 50, 0.1);

/* 按钮悬停渐变 */
background: linear-gradient(135deg, #C93832, #E04444);
```

## 三、交互动效建议

### 1. 进度条动画

```css
/* 进度条流动效果 */
@keyframes progress-flow {
  0% { background-position: 0 0; }
  100% { background-position: 30px 0; }
}

.progress-bar {
  background: linear-gradient(90deg, 
    #C93832 0%, #FF6B6B 50%, #C93832 100%);
  background-size: 30px 100%;
  animation: progress-flow 1s linear infinite;
}
```

### 2. Agent状态切换

| 动效 | 描述 | 时长 |
|------|------|------|
| 脉冲 | 运行中状态，呼吸灯效果 | 2s循环 |
| 弹入 | 新任务分配，卡片弹入 | 0.3s |
| 闪烁 | 错误状态，警示闪烁 | 0.5s |

```css
/* Agent运行中脉冲效果 */
@keyframes pulse {
  0%, 100% { 
    box-shadow: 0 0 0 0 rgba(0, 214, 143, 0.4);
  }
  50% { 
    box-shadow: 0 0 0 10px rgba(0, 214, 143, 0);
  }
}

.agent-running {
  animation: pulse 2s infinite;
}
```

### 3. 日志滚动

- **自动滚动**：新日志出现时自动滚到底部
- **高亮新条目**：新日志淡入效果（0.3s）
- **颜色编码**：不同级别日志不同颜色

```css
.log-entry {
  animation: fadeIn 0.3s ease-out;
}

@keyframes fadeIn {
  from { 
    opacity: 0; 
    transform: translateY(-10px); 
  }
  to { 
    opacity: 1; 
    transform: translateY(0); 
  }
}
```

### 4. 卡片悬停

```css
.card {
  transition: all 0.3s ease;
}

.card:hover {
  transform: translateY(-2px);
  box-shadow: 0 8px 30px rgba(201, 56, 50, 0.2);
  border-color: rgba(201, 56, 50, 0.5);
}
```

### 5. 数据刷新

- **数字跳动**：统计数字更新时跳动效果
- **波纹扩散**：新任务到达时波纹提示
- **平滑过渡**：进度变化平滑动画
