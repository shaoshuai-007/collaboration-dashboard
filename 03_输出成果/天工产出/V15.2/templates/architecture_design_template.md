#### 5.1.1 URL规范
- 基础路径: `/api/v1/`
- 资源命名: 使用名词复数形式
- 版本控制: URL中包含版本号

#### 5.1.2 请求/响应格式

**请求格式**:
```json
{
  "requestId": "uuid",
  "timestamp": "ISO8601",
  "data": {}
}
```

**响应格式**:
```json
{
  "code": 200,
  "message": "success",
  "data": {},
  "timestamp": "ISO8601"
}
```

### 5.2 核心API列表
