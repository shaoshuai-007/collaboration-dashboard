## 二、思维链推导

### 第一层：理解她的立场

```
张艳50岁 → 政治觉悟高 → 推崇集团战略 → 关停并转是大势
         → 她不会支持"逆势保项目"
         → 她会用"集团趋势"来堵你的嘴
         → 她会用"你们没卡住位"来质疑你的能力
```

**结论**：她的质疑是"政治正确"的保护色，也是对过往执行力的失望。

### 第二层：找到破局点

```
破局点一：不是"逆势保项目"，而是"顺势转型"
         → 关停并转不可逆 → 我们认同
         → 但"转"不是停 → 转到哪里？转给谁？
         → 新东网占的是"存量"，我们切入的是"增量"

破局点二：竞争对手存在 ≠ 没有机会
         → 新东网有平台，但平台有"边际效应递减"
         → 长期合作 → 客户可能想引入竞争 → 防止供应商绑定
         → AI是新赛道，新东网的平台不一定能跟上AI化
```

**结论**：不争存量，争增量；不逆大势，顺势而为。

### 第三层：制定策略

| 策略 | 核心逻辑 | 话术方向 |
|------|---------|---------|
| **策略一：顺势认同** | 先认同她的"政治正确"，降低防御 | "张经理说得对，集团趋势不可逆，我们不逆势" |
| **策略二：承认不足** | 承认过去卡位不力，但不纠缠过去 | "前两年确实没卡住，这个我们反思" |
| **策略三：转化机会** | 竞争对手存在=引入新供应商的机会 | "新东网确实占了平台，但AI是新赛道，我们可以互补" |
| **策略四：增量切入** | 不争存量，争增量价值 | "我们不抢新东网的存量，我们切入AI增量" |

## 三、场景应对脚本

### 场景一：她抛出"集团趋势不会变"

**她的原话（预判）**：
> "集团的趋势不会变，项目收紧是大势所趋，电渠项目要顺势而为，不要逆势。"

**应对脚本**：

```
【第一步：认同政治正确】

"张经理说得对，集团趋势不可逆，关停并转是大方向。
我们不逆势，也逆不了势。"

【第二步：转化"转"的方向】

"但'转'不是'停'，关键是怎么转。
电渠的用户不能断，业务要平稳过渡。
我们想的是——停变为转，转型中国电信APP，
让电渠的价值在新的平台上延续。"

【第三步：提出我们的价值】

"我们不是来保存量项目的，而是帮您想好'转'的路径。
转型需要的智能化工具、用户承接方案，我们可以支撑。"
```

**话术速记**：
> **"集团趋势我们认同，但'转'需要路径，我们帮您想好怎么转。"**

---

### 场景二：她抛出"新东网已经占了，你们前两年没卡住位"

**她的原话（预判）**：
> "前两年和你们说了卡位，但电渠项目基本是竞争对手新东网占着，他们有订单管理中心、数字化运营平台、企微平台。你们现在来，晚了。"

**应对脚本**：

```
【第一步：承认不足，不纠缠过去】

"张经理说得对，前两年我们确实没卡住位，
这个我们反思，执行力不够。
过去的错，我们认。"

【第二步：转化竞争格局】

"但换个角度，新东网占的是'平台存量'，
AI是新的赛道，他们不一定能跟上。

我们观察到，新东网的平台主要做的是流程管理，
但AI赋能、智能化运营，这是新的增量，
这块我们可以互补，不冲突。"

【第三步：提出客户价值】

"而且，供应商单一化对省公司也有风险。
引入第二供应商，形成竞争格局，
对新东网也是一种倒逼，对省公司是有利的。

我们不是来抢存量，是来做增量，
AI智能化这块，我们可以帮您补位。"
```

**话术速记**：
> **"新东网占的是平台存量，我们切入的是AI增量，互补不冲突。"**

---

### 场景三：她提出"预算收紧，没有空间"

**她的原话（预判）**：
> "现在预算收紧，电渠项目在关停并转名单里，没有预算空间给你们。"

**应对脚本**：

```
【第一步：理解她的难处】

"理解，预算确实是硬约束，我们不强求新增预算。"

【第二步：提出预算结构调整】

"但能不能换个思路？
关停WAP厅后，原有的人力、预算是否可以结构调整？
不是新增，是存量转移。

比如：
- WAP厅关停后，人力转向APP运营智能化
- 原有预算调整为AI工具采购
- 总盘子不变，但价值更高"

【第三步：提出合作模式】

"我们可以先做个小范围试点，
用最小的预算验证AI增量价值，
效果好再谈扩大，风险可控。"
```

**话术速记**：
> **"不是新增预算，是存量结构调整，总盘子不变，价值更高。"**
