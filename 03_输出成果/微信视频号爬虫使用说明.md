# 微信视频号爬虫工具 - 使用说明

## 🌿 工具概述

南乔为少帅开发的微信视频号爬虫工具，支持自动登录、搜索账号、获取视频列表。

## 📦 安装依赖

```bash
cd /root/.openclaw/workspace
source .venv/bin/activate
pip install DrissionPage
```

## 🚀 使用方法

### 方法1：自动爬取（推荐）

```bash
cd /root/.openclaw/workspace
source .venv/bin/activate
python scripts/wechat_channels_crawler_final.py
```

**首次运行**：
1. 脚本会打开浏览器
2. 显示微信登录二维码
3. 使用手机微信扫码登录
4. 登录成功后自动保存登录态
5. 搜索目标账号并爬取

**后续运行**：
- 自动使用保存的登录态
- 无需重复扫码

### 方法2：手动方案

如果自动爬取失败，可以手动操作：

1. **手机录屏**
   - 在手机上打开微信视频号
   - 找到目标视频并播放
   - 使用手机录屏功能录制

2. **语音转文字**
   - 使用"通义听悟"、"讯飞听见"等工具
   - 将录屏中的语音转成文字

3. **整理成文档**
   - 将转录内容整理成PPT或Word

### 方法3：第三方服务

使用专业的数据服务平台：

- **新榜** - 微信生态数据服务
- **清博大数据** - 社交媒体数据分析
- **蝉妈妈** - 视频号数据分析平台

## ⚠️ 注意事项

### 1. 浏览器要求

- 需要安装Chrome或Chromium浏览器
- 确保浏览器版本较新

### 2. 微信限制

微信视频号有严格的内容保护机制：

- **网页版功能受限**：只能查看部分内容
- **需要登录态**：大部分功能需要登录
- **动态渲染**：内容通过JavaScript加载
- **反爬机制**：可能触发风控

### 3. 法律合规

爬取数据需遵守：

- 微信平台协议
- 数据安全法规
- 个人信息保护法
- 仅用于学习研究

## 🔧 常见问题

### Q1: 浏览器启动失败？

**原因**：未安装Chrome浏览器

**解决**：
```bash
# 安装Chrome
wget https://dl.google.com/linux/direct/google-chrome-stable_current_amd64.deb
sudo apt install ./google-chrome-stable_current_amd64.deb
```

### Q2: 登录超时？

**原因**：未在5分钟内完成扫码

**解决**：
- 确保手机网络正常
- 重新运行脚本

### Q3: 找不到视频？

**原因**：
- 账号名称不准确
- 微信网页版功能受限

**解决**：
- 确认账号名称正确
- 尝试手动方案

### Q4: 内容提取失败？

**原因**：页面结构与预期不符

**解决**：
- 查看保存的HTML文件
- 手动调整选择器
