## 二、问题诊断

### 2.1 核心问题清单

| 问题 | 现状 | 影响 | 严重程度 |
|------|------|------|:--------:|
| **P1: 无客户意图识别** | 识别的是"任务类型"（REQ-02），不是"客户意图"（做什么） | Agent不知道客户真正想要什么 | 🔴高 |
| **P2: 单轮发言** | 每个Agent只发言一次 | 缺乏自由交流、补充、碰撞 | 🔴高 |
| **P3: 无真正互动** | Agent按固定顺序发言，不互相回应 | 讨论流于形式 | 🔴高 |
| **P4: 共识计算机械** | 70% - 质疑数×5% | 不能反映真正共识 | 🟡中 |
| **P5: 无会议纪要** | 只有简单总结 | 无法追溯决策过程 | 🔴高 |
| **P6: 产出物不具体** | 只输出模板名称 | 不知道后续输出什么 | 🟡中 |

### 2.2 问题根因分析

#### 问题1：无客户意图识别

**现象**：
```
用户输入："做一个智能客服系统"
↓
当前识别：REQ-02 (需求分析)
↓
Agent收到：task = "做一个智能客服系统"
↓
问题：Agent不知道客户想要几个产出物、什么深度
```

**根因**：
- 意图识别只识别"任务类型"，不识别"客户意图"
- 缺少南乔"理解+确认"环节
- 没有把模糊需求转化为明确的产出物清单

#### 问题2：单轮发言

**现象**：
```
采薇发言1次 → 结束
织锦发言1次 → 结束
玉衡发言1次 → 结束
...
```

**根因**：
```python
# 代码只循环一轮
for agent_key in discussion_flow:
    response = responder.generate(agent, task, memory)
    memory.add_turn(...)
# 循环结束，讨论结束
```

- 只有单层循环，没有多轮机制
- 没有"检测到冲突→继续讨论"的逻辑

#### 问题3：无真正互动

**现象**：
```
采薇：需求包括A、B、C...
织锦：技术方案是X、Y、Z...  （没有回应采薇的观点）
玉衡：风险有1、2、3...      （没有回应织锦的观点）
```

**根因**：
```python
# Agent虽然能看到context，但prompt没有要求回应
user_message = f"""当前任务：{task}
已有讨论：{context}
{role_prompt}。如果有不同意见请明确提出质疑。"""
```

- prompt只说"有不同意见提出质疑"，没有强制要求回应
- Agent倾向于发表自己的观点，不回应别人

## 三、目标对齐

### 3.1 目标拆解

```
目标：各角色针对客户意图识别后进行专业分析，自由交流、补充、碰撞，
      最终达成一致结论，给出后续输出内容方向，形成完整会议纪要
```

拆解为6个子目标：

| 子目标 | 含义 | 当前状态 |
|--------|------|:--------:|
| 客户意图识别 | 理解客户想要什么 | ❌缺失 |
| 各自专业分析 | 每个角色从专业角度分析 | ✅已有 |
| 自由交流 | Agent之间多轮对话 | ❌缺失 |
| 补充碰撞 | 质疑、反驳、完善 | ⚠️部分有 |
| 达成一致结论 | 共识输出 | ⚠️机械计算 |
| 后续输出方向 | 明确产出物清单 | ❌缺失 |
| 完整会议纪要 | 文档化输出 | ❌缺失 |

### 3.2 场景分析

#### 场景1：单一产出物

用户："做个需求分析"

产出物：需求规格说明书 × 1
参与Agent：采薇(主导) + 织锦 + 玉衡
讨论轮次：1-2轮
```

#### 场景2：多产出物（链式）
```

产出物：需求规格说明书 + 系统架构设计 × 2
参与Agent：采薇 + 织锦(主导) + 玉衡

#### 场景3：全流程（多产出物）
```

产出物：需求 + 架构 + 详细设计 + 项目计划 + 成本估算 × 5

依赖关系：需求 → 架构 → 详细设计，需求+架构 → 项目计划 → 成本
```

#### 场景4：模糊需求
```
用户："帮我整理一下想法"
↓
南乔理解：需要识别客户到底想要什么
确认流程：南乔反馈 → 用户确认 → 开始讨论
```

### 4.2 核心模块设计

#### 模块1：南乔意图分析器

```python
class NanqiaoIntentAnalyzer:
    """南乔意图分析器 - 理解客户真正想要什么"""
    
    def analyze(self, user_input: str) -> dict:
        """
        分析用户意图
        
        Returns:
            {
                'project_name': '智能客服系统',
                'task_type': 'REQ-02',
                'task_name': '需求分析',
                'outputs': [
                    {'name': '需求规格说明书', 'code': 'SRS', 'owner': '采薇'}
                ],
                'complexity': 'medium',
                'participants': ['采薇', '织锦', '玉衡', '呈彩', '工尺'],
                'discussion_rounds': 2,
                'structured_intent': '为智能客服系统进行需求分析，输出需求规格说明书'
            }
        """
        # 1. 提取项目名称
        project_name = self._extract_project_name(user_input)
        
        # 2. 识别产出物
        outputs = self._identify_outputs(user_input)
        
        # 3. 映射任务类型
        task_type = self._map_to_task_type(outputs)
        
        # 4. 判断复杂度
        complexity = self._assess_complexity(outputs)
        
        # 5. 确定参与角色
        participants = self._determine_participants(task_type, complexity)
        
        # 6. 计算讨论轮次
        discussion_rounds = self._calculate_rounds(complexity, len(participants))
        
        # 7. 生成结构化意图
        structured_intent = self._generate_structured_intent(
            project_name, task_type, outputs
        )
        
        return {
            'project_name': project_name,
            'task_type': task_type,
            'task_name': TASK_TYPES[task_type].name,
            'outputs': outputs,
            'complexity': complexity,
            'participants': participants,
            'discussion_rounds': discussion_rounds,
            'structured_intent': structured_intent
        }
    
    def _identify_outputs(self, user_input: str) -> list:
        """识别产出物"""
        output_keywords = {
            '需求': {'name': '需求规格说明书', 'code': 'SRS', 'owner': '采薇'},
            '需求文档': {'name': '需求规格说明书', 'code': 'SRS', 'owner': '采薇'},
            'SRS': {'name': '需求规格说明书', 'code': 'SRS', 'owner': '采薇'},
            '架构': {'name': '系统架构设计文档', 'code': 'SAD', 'owner': '织锦'},
            '架构图': {'name': '系统架构设计文档', 'code': 'SAD', 'owner': '织锦'},
            '系统设计': {'name': '系统架构设计文档', 'code': 'SAD', 'owner': '织锦'},
            '详细设计': {'name': '详细设计说明书', 'code': 'DDD', 'owner': '工尺'},
            '接口设计': {'name': '详细设计说明书', 'code': 'DDD', 'owner': '工尺'},
            '项目计划': {'name': '项目计划书', 'code': 'PP', 'owner': '扶摇'},
            '排期': {'name': '项目计划书', 'code': 'PP', 'owner': '扶摇'},
            '成本': {'name': '成本估算表', 'code': 'CE', 'owner': '筑台'},
            '预算': {'name': '成本估算表', 'code': 'CE', 'owner': '筑台'},
            '报价': {'name': '成本估算表', 'code': 'CE', 'owner': '筑台'},
            '风险': {'name': '风险登记册', 'code': 'RR', 'owner': '玉衡'},
            '完整方案': 'ALL',  # 特殊标记：全流程
            '可落地': 'ALL',
            '全流程': 'ALL',
        }
        
        outputs = []
        has_all = False
        
        for keyword, output_info in output_keywords.items():
            if keyword in user_input:
                if output_info == 'ALL':
                    has_all = True
                    break
                outputs.append(output_info)
        
        # 全流程：6个产出物
        if has_all:
            outputs = [
                {'name': '需求规格说明书', 'code': 'SRS', 'owner': '采薇'},
                {'name': '系统架构设计文档', 'code': 'SAD', 'owner': '织锦'},
                {'name': '详细设计说明书', 'code': 'DDD', 'owner': '工尺'},
                {'name': '项目计划书', 'code': 'PP', 'owner': '扶摇'},
                {'name': '成本估算表', 'code': 'CE', 'owner': '筑台'},
                {'name': '风险登记册', 'code': 'RR', 'owner': '玉衡'},
            ]
        
        # 去重
        seen = set()
        unique_outputs = []
        for o in outputs:
            if o['code'] not in seen:
                seen.add(o['code'])
                unique_outputs.append(o)
        
        return unique_outputs if unique_outputs else [
            {'name': '需求规格说明书', 'code': 'SRS', 'owner': '采薇'}
        ]

class MultiRoundDiscussion:
    """多轮讨论执行器"""
    
    def __init__(self, max_rounds: int = 3):
        self.max_rounds = max_rounds
        self.discussion_history = []
        self.pending_replies = []  # 待回应的质疑
    
    def run(self, task: str, participants: list, structured_intent: dict) -> dict:
        """
        执行多轮讨论
        
        Returns:
            {
                'rounds': [...],
                'consensus_reached': True/False,
                'final_consensus': '...',
                'key_decisions': [...]
            }
        """
        rounds = []
        consensus_reached = False
        
        for round_num in range(1, self.max_rounds + 1):
            round_result = self._run_round(
                round_num, task, participants, structured_intent
            )
            rounds.append(round_result)
            
            # 检测是否达成共识
            if self._check_consensus(round_result):
                consensus_reached = True
                break
            
            # 检测是否有待回应的质疑
            if not self._has_pending_challenges(round_result):
                consensus_reached = True
                break
        
        return {
            'rounds': rounds,
            'consensus_reached': consensus_reached,
            'final_consensus': self._generate_consensus(rounds),
            'key_decisions': self._extract_decisions(rounds)
        }
    
    def _run_round(self, round_num: int, task: str, participants: list, 
                   structured_intent: dict) -> dict:
        """执行单轮讨论"""
        round_messages = []
        
        for agent_key in participants:
            # 生成Agent发言
            response = self._generate_agent_response(
                agent_key, task, round_num, structured_intent
            )
            
            # 检测是否质疑
            is_challenge = self._detect_challenge(response)
            
            # 检测是否回应
            reply_to = self._detect_reply(response)
            
            round_messages.append({
                'round': round_num,
                'speaker': agent_key,
                'content': response,
                'is_challenge': is_challenge,
                'reply_to': reply_to
            })
        
        return {
            'round_num': round_num,
            'messages': round_messages,
            'challenge_count': sum(1 for m in round_messages if m['is_challenge'])
        }
    
    def _generate_agent_response(self, agent_key: str, task: str, 
                                  round_num: int, structured_intent: dict) -> str:
        """生成Agent发言（带互动逻辑）"""
        
        agent = AGENTS[agent_key]
        context = self._get_context_for_round(round_num)
        
        # 根据轮次生成不同的prompt
        if round_num == 1:
            # 第一轮：专业分析
            prompt = f"""当前任务：{structured_intent['structured_intent']}

请从你的专业角度（{agent.role}）进行分析：
1. 核心观点
2. 关键要点
3. 风险提示（如有）

要求：观点明确，逻辑清晰。"""
        
        elif round_num == 2:
            # 第二轮：补充碰撞
            prompt = f"""当前任务：{structured_intent['structured_intent']}

已有讨论：
{context}

请根据上述讨论：
1. 如果你有不同意见，请明确指出并说明理由
2. 如果有需要补充的观点，请补充
3. 如果认为方案可行，请确认

要求：必须回应前面发言者的观点，形成真正的讨论。"""
        
        else:
            # 第三轮：共识确认
            prompt = f"""当前任务：{structured_intent['structured_intent']}

讨论历史：
{context}

请确认：
1. 是否还有未解决的问题？
2. 最终方案是什么？
3. 需要后续跟进的事项？"""
        
        response = call_qianfan(agent.system_prompt, prompt)
        return response
    
    def _detect_challenge(self, response: str) -> bool:
        """检测是否质疑"""
        challenge_keywords = [
            '质疑', '反对', '不同意', '不可行', 
            '成本过高', '周期太长', '有风险',
            '我认为不对', '这个方案有问题'
        ]
        return any(kw in response for kw in challenge_keywords)
    
    def _detect_reply(self, response: str) -> str:
        """检测回应的是谁"""
        # 检测response中是否提到了某个Agent
        for agent_key, agent in AGENTS.items():
            if agent.name in response:
                return agent.name
        return ""

class MultiOutputWorkflow:
    """多产出物工作流"""
    
    # 产出物依赖关系
    OUTPUT_DEPENDENCIES = {
        'SRS': [],                    # 需求文档 - 无依赖
        'SAD': ['SRS'],               # 架构设计 - 依赖需求
        'DDD': ['SRS', 'SAD'],        # 详细设计 - 依赖需求+架构
        'PP': ['SRS', 'SAD'],         # 项目计划 - 依赖需求+架构
        'CE': ['PP'],                 # 成本估算 - 依赖项目计划
        'RR': ['SRS', 'SAD'],         # 风险登记 - 依赖需求+架构
    }
    
    def execute(self, outputs: list, project_name: str) -> dict:
        """执行多产出物工作流"""
        
        # 1. 按依赖关系排序
        ordered_outputs = self._topological_sort(outputs)
        
        # 2. 分阶段执行
        results = {}
        for output in ordered_outputs:
            # 获取依赖的产出物
            dependencies = self._get_dependencies(output, results)
            
            # 执行讨论
            discussion = self._run_discussion(output, project_name, dependencies)
            
            # 记录结果
            results[output['code']] = discussion
        
        # 3. 生成完整会议纪要
        full_minutes = self._generate_full_minutes(results)
        
        return {
            'outputs': results,
            'meeting_minutes': full_minutes
        }
    
    def _topological_sort(self, outputs: list) -> list:
        """拓扑排序"""
        # 按依赖关系排序
        sorted_outputs = []
        added_codes = set()
        
        while len(sorted_outputs) < len(outputs):
            for output in outputs:
                if output['code'] in added_codes:
                    continue
                
                # 检查依赖是否已添加
                deps = self.OUTPUT_DEPENDENCIES.get(output['code'], [])
                if all(d in added_codes for d in deps):
                    sorted_outputs.append(output)
                    added_codes.add(output['code'])
        
        return sorted_outputs
```
