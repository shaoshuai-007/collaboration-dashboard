### 任务22：成本估算模型（P2）

**目标**：参数化成本测算

**内容**：
```python
class CostEstimator:
    def estimate(self, params):
        """
        params = {
            'team_size': 5,        # 团队规模
            'duration_months': 6,  # 项目周期
            'complexity': '中',     # 复杂度
            'tech_stack': ['Java', 'Vue', 'MySQL'],
            'cloud_service': True,
        }
        """
        # 人天计算
        man_days = self._calc_man_days(params)
        
        # 人力成本
        labor_cost = man_days * 1500  # 元/人天
        
        # 云服务成本
        cloud_cost = self._calc_cloud_cost(params)
        
        # 风险储备（10%）
        risk_reserve = (labor_cost + cloud_cost) * 0.1
        
        # 总成本
        total = labor_cost + cloud_cost + risk_reserve
        
        return {
            'man_days': man_days,
            'labor_cost': labor_cost,
            'cloud_cost': cloud_cost,
            'risk_reserve': risk_reserve,
            'total': total,
        }
```

**产出**：
- 成本估算接口
- 报价单模板
- 参数输入界面

**工时**：2天

### 任务23：流式输出（P1）

**目标**：Agent发言实时显示

**技术方案**：
```python
# 后端：SSE（Server-Sent Events）
from flask import Response

@app.route('/api/stream/<task_id>')
def stream_task(task_id):
    def generate():
        for chunk in execute_task_stream(task_id):
            yield f"data: {json.dumps(chunk)}\n\n"
    return Response(generate(), mimetype='text/event-stream')

# 前端：EventSource
const eventSource = new EventSource('/api/stream/task_001');
eventSource.onmessage = (event) => {
    const data = JSON.parse(event.data);
    appendMessage(data.agent, data.content);
};
```
